1.2 计算机如何工作？——IPO模型与算法思维

核心知识点

IPO模型：计算机的“工作三步曲”

计算机就像一个聪明的“助手”，它的工作方式非常清晰，可以分为三个基本步骤：

📥
Input（输入）

你给计算机提供信息（比如数字、名字、选择）

⚙️
Process（处理）

计算机根据你写的程序，对这些信息进行计算、判断或操作

📤
Output（输出）

计算机把处理结果告诉你（比如显示在屏幕上）

这个过程叫做 IPO模型，是所有程序的基础。

生活中的例子：

  • 做果汁：
    • 输入：水果、水、糖
    • 处理：用榨汁机搅拌
    • 输出：一杯果汁
  • 考试：
    • 输入：试卷和答题卡
    • 处理：老师批改
    • 输出：你的分数

算法：解决问题的“菜谱”

算法 就是解决问题的一系列明确、有序的步骤。就像做菜的菜谱一样，每一步都要写清楚，计算机才能照着做。

  • 算法必须是具体的（不能说“炒一下”，要说“炒30秒”）
  • 算法必须有开始和结束
  • 同一个问题可以有不同的算法

例子：如何从家走到学校？

  1. 走出家门
  2. 向右走到路口
  3. 看红绿灯，如果是绿灯，过马路
  4. 直走200米，看到学校大门
  5. 进入学校
  6. 结束

流程图：用图形画出算法

流程图 是用图形和箭头来表示算法的工具。它比文字更直观，能帮助我们理清思路。

常用图形：

  • 椭圆形：开始 / 结束
  • 平行四边形：输入 / 输出
  • 矩形：处理（计算、赋值）
  • 菱形：判断（是/否）
  • 箭头：流程方向

代码样例：用C++实现一个IPO程序

我们来写一个程序，判断一个数是奇数还是偶数。

#include <iostream>
using namespace std;

int main() {
    // ========== INPUT（输入）==========
    int number;
    cout << "请输入一个整数：";
    cin >> number;

    // ========== PROCESS（处理）==========
    // 判断是否为偶数：能被2整除就是偶数
    if (number % 2 == 0) {
        // ========== OUTPUT（输出）==========
        cout << number << " 是偶数。" << endl;
    } else {
        // ========== OUTPUT（输出）==========
        cout << number << " 是奇数。" << endl;
    }

    return 0;
}

代码解释：

  • cin >> number; → 输入
  • if (number % 2 == 0) → 处理（判断是否能被2整除）
  • cout << ... → 输出
