条件嵌套和三目运算

学习如何使用条件嵌套和三目运算符来简化代码逻辑

条件嵌套

条件嵌套是指在一个条件语句中嵌套另一个条件语句，用于处理更复杂的逻辑。

#include <iostream>
using namespace std;

int main() {
    int score;
    cout << "请输入分数: ";
    cin >> score;

    if (score >= 60) {
        if (score >= 90) {
            cout << "优秀" << endl;
        } else {
            cout << "及格" << endl;
        }
    } else {
        cout << "不及格" << endl;
    }

    return 0;
}

三目运算符

三目运算符（Ternary Operator）是条件判断的简写形式，语法如下：

条件表达式 ? 表达式1 : 表达式2;

如果条件表达式为真，执行表达式1；否则执行表达式2。

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;
    int max = (a > b) ? a : b;

    cout << "较大的值是: " << max << endl;
    return 0;
}

总结

条件嵌套和三目运算符是控制程序逻辑的重要工具。条件嵌套适合处理复杂逻辑，而三目运算符可以让简单的条件判断更加简洁。

练习：尝试编写一个程序，输入三个数字，输出其中最大的值。
