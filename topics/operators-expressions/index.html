基础运算符

算术运算符

用于执行基本的数学运算：

运算符 名称 示例 说明
+ 加法 a + b 将两个数相加
- 减法 a - b 将两个数相减
* 乘法 a * b 将两个数相乘
/ 除法 a / b 将两个数相除
% 取模 a % b 求两个数相除的余数
++ 自增 ++a 或 a++ 将变量值增加1
-- 自减 --a 或 a-- 将变量值减少1
int a = 10, b = 3;
cout << "a + b = " << a + b << endl; // 输出: 13
cout << "a - b = " << a - b << endl; // 输出: 7
cout << "a * b = " << a * b << endl; // 输出: 30
cout << "a / b = " << a / b << endl; // 输出: 3 (整数除法)
cout << "a % b = " << a % b << endl; // 输出: 1 (余数)

关系运算符

用于比较两个值之间的关系，结果为布尔值(true/false)：

运算符 名称 示例 说明
== 等于 a == b 判断两个值是否相等
!= 不等于 a != b 判断两个值是否不相等
> 大于 a > b 判断左边值是否大于右边值
< 小于 a < b 判断左边值是否小于右边值
>= 大于等于 a >= b 判断左边值是否大于等于右边值
<= 小于等于 a <= b 判断左边值是否小于等于右边值
int x = 5, y = 8;
cout << (x == y) << endl; // 输出: 0 (false)
cout << (x != y) << endl; // 输出: 1 (true)
cout << (x < y) << endl; // 输出: 1 (true)
cout << (x >= y) << endl; // 输出: 0 (false)

高级运算符

逻辑运算符

用于组合多个条件表达式：

运算符 名称 示例 说明
&& 逻辑与 a && b 两个条件都为真时结果为真
|| 逻辑或 a || b 两个条件至少一个为真时结果为真
! 逻辑非 !a 将条件的结果取反

逻辑运算符的运算规则

逻辑运算符的运算规则遵循布尔代数原理：

  • 逻辑与(&&)：只有当两个操作数都为真时，结果才为真
  • 逻辑或(||)：只要有一个操作数为真，结果就为真
  • 逻辑非(!)：将操作数的真假性取反
bool a = true, b = false;

// 逻辑与运算
cout << "a && a = " << (a && a) << endl; // 输出: 1 (true)
cout << "a && b = " << (a && b) << endl; // 输出: 0 (false)
cout << "b && a = " << (b && a) << endl; // 输出: 0 (false)
cout << "b && b = " << (b && b) << endl; // 输出: 0 (false)

// 逻辑或运算
cout << "a || a = " << (a || a) << endl; // 输出: 1 (true)
cout << "a || b = " << (a || b) << endl; // 输出: 1 (true)
cout << "b || a = " << (b || a) << endl; // 输出: 1 (true)
cout << "b || b = " << (b || b) << endl; // 输出: 0 (false)

// 逻辑非运算
cout << "!a = " << !a << endl; // 输出: 0 (false)
cout << "!b = " << !b << endl; // 输出: 1 (true)

逻辑运算符的短路求值

逻辑运算符具有短路求值特性，这是一种重要的优化机制：

短路求值规则

  • 逻辑与(&&)：如果左侧表达式为假(false)，则不会计算右侧表达式
  • 逻辑或(||)：如果左侧表达式为真(true)，则不会计算右侧表达式

短路求值示例

int a = 5, b = 0;
bool result;

// 逻辑与短路示例
// 由于 b != 0 为假(0)，不会计算 a / b，避免了除零错误
result = (b != 0) && (a / b > 1);
cout << "(b != 0) && (a / b > 1) = " << result << endl;

// 逻辑或短路示例
// 由于 a == 5 为真(1)，不会计算 b / (a-5)，避免了除零错误
result = (a == 5) || (b / (a-5) > 0);
cout << "(a == 5) || (b / (a-5) > 0) = " << result << endl;

短路求值不仅可以提高程序效率，还能避免一些潜在的错误。

赋值运算符

用于将值赋给变量：

运算符 示例 等价表达式
= a = b a = b
+= a += b a = a + b
-= a -= b a = a - b
*= a *= b a = a * b
/= a /= b a = a / b
%= a %= b a = a % b

赋值运算符的特点

赋值运算符具有以下特点：

  • 右结合性：赋值运算符从右向左结合，如 a = b = c 等价于 a = (b = c)
  • 返回值：赋值表达式会返回被赋的值，可以用于连续赋值
  • 复合赋值：复合赋值运算符是简化表达式的语法糖
int x, y, z;

// 连续赋值
x = y = z = 10;
cout << "x = " << x << ", y = " << y << ", z = " << z << endl;

// 复合赋值运算符的使用
int a = 20;
a += 5; // 等价于 a = a + 5; 结果 a = 25
cout << "a += 5 后，a = " << a << endl;

a *= 2; // 等价于 a = a * 2; 结果 a = 50
cout << "a *= 2 后，a = " << a << endl;

a /= 5; // 等价于 a = a / 5; 结果 a = 10
cout << "a /= 5 后，a = " << a << endl;

条件运算符（三元运算符）

条件运算符是C++中唯一的三元运算符，语法为：条件 ? 值1 : 值2

由于我们尚未学习分支语句，这里仅作简单介绍，后续章节会详细讲解。

条件运算符的工作原理

条件运算符根据条件的真假性来选择返回值：

  • 如果条件为真(true)，则返回值1
  • 如果条件为假(false)，则返回值2
int a = 10, b = 20;
int max = (a > b) ? a : b; // 如果a>b为真，返回a，否则返回b
cout << "较大的数是: " << max << endl;

// 嵌套使用条件运算符
int x = 5, y = 10, z = 3;
int min = (x < y) ? ((x < z) ? x : z) : ((y < z) ? y : z);
cout << "最小值是: " << min << endl;

表达式

表达式的定义

表达式是由操作数（变量、常量）和运算符组成的组合，它会产生一个值。

// 以下都是表达式
5; // 常量表达式
x; // 变量表达式
x + y; // 算术表达式
x > 0; // 关系表达式
(x + y) * z; // 复合表达式
x = y = 5; // 赋值表达式

表达式的求值过程

表达式的求值是一个逐步进行的过程，遵循运算符优先级和结合性规则：

求值步骤

  1. 确定运算符优先级：找出表达式中优先级最高的运算符
  2. 考虑结合性：如果有多个相同优先级的运算符，根据结合性确定计算顺序
  3. 逐步计算：从优先级最高的运算开始，逐步计算整个表达式

复杂表达式求值示例

让我们分析表达式：a + b * c - d / e

int a = 10, b = 5, c = 3, d = 8, e = 2;
int result = a + b * c - d / e;
// 求值过程：
// 1. * 和 / 优先级最高，从左到右计算：
// b * c = 5 * 3 = 15
// d / e = 8 / 2 = 4
// 2. 表达式变为：a + 15 - 4
// 3. + 和 - 优先级相同，从左到右计算：
// a + 15 = 10 + 15 = 25
// 25 - 4 = 21
// 4. 最终结果：21
cout << "a + b * c - d / e = " << result << endl;

使用括号控制求值顺序

括号可以改变默认的求值顺序，使表达式更加清晰：

// 不使用括号（按优先级计算）
result = a + b * c; // 先算乘法，再算加法

// 使用括号改变顺序
result = (a + b) * c; // 先算加法，再算乘法

运算符优先级

当表达式中有多个运算符时，运算符优先级决定了计算顺序：

优先级 运算符 说明 结合性
1 () [] -> . ++ -- 括号、数组下标、成员访问、自增自减 从左到右
2 ! ~ ++ -- + - * & 逻辑非、位取反、正负号、取地址、解引用 从右到左
3 * / % 乘法、除法、取模 从左到右
4 + - 加法、减法 从左到右
5 < <= > >= 关系运算符 从左到右
6 == != 相等运算符 从左到右
7 && 逻辑与 从左到右
8 || 逻辑或 从左到右
9 ?: 条件运算符 从右到左
10 = += -= *= /= %= 赋值运算符 从右到左

优先级示例

int result = 5 + 3 * 2; // 结果是 11，不是 16
// 因为 * 的优先级高于 +，所以先计算 3 * 2 = 6，再计算 5 + 6 = 11

int result2 = (5 + 3) * 2; // 结果是 16
// 使用括号改变优先级

运算符优先级详解

运算符优先级决定了表达式中各个运算的执行顺序。以下是一些重要规则：

优先级规则

  1. 括号优先：括号具有最高优先级，可以改变默认的运算顺序
  2. 算术运算符：乘法(*)、除法(/)、取模(%)优先级高于加法(+)、减法(-)
  3. 关系运算符：关系运算符(<、>、<=、>=、==、!=)优先级低于算术运算符
  4. 逻辑运算符：逻辑与(&&)优先级高于逻辑或(||)
  5. 赋值运算符：赋值运算符(=、+=、-=等)优先级最低

结合性

当表达式中出现多个相同优先级的运算符时，结合性决定了它们的计算顺序：

  • 左结合：大多数运算符从左到右计算，如 a + b + c 等价于 (a + b) + c
  • 右结合：赋值运算符从右到左计算，如 a = b = c 等价于 a = (b = c)

优先级示例解析

int a = 5, b = 3, c = 2;
int result = a + b * c; // 结果是 11，不是 16
// 解析过程：
// 1. * 运算符优先级高于 +，先计算 b * c = 3 * 2 = 6
// 2. 再计算 a + 6 = 5 + 6 = 11
cout << "a + b * c = " << result << endl;

result = (a + b) * c; // 结果是 16
// 解析过程：
// 1. 括号优先级最高，先计算 (a + b) = (5 + 3) = 8
// 2. 再计算 8 * c = 8 * 2 = 16
cout << "(a + b) * c = " << result << endl;

类型转换

在表达式中不同类型的数据参与运算时会发生类型转换：

int a = 10;
double b = 3.0;
double result = a / b; // int类型的a会转换为double类型
cout << result << endl; // 输出: 3.33333

int c = 10;
int d = 3;
double result2 = c / d; // 先计算10/3=3(整数除法)，再转换为double
cout << result2 << endl; // 输出: 3

double result3 = (double)c / d; // 强制类型转换
cout << result3 << endl; // 输出: 3.33333

类型转换详解

在表达式计算过程中，不同类型的数据可能需要进行转换以保证运算的正确性。

自动类型转换

C++会在必要时自动进行类型转换：

  • 整型转浮点型：当整数与浮点数进行运算时，整数会自动转换为浮点数
  • 低精度转高精度：当不同类型的数据进行运算时，低精度类型会转换为高精度类型

自动类型转换示例

int a = 10;
double b = 3.0;

// 整型a自动转换为double类型参与运算
double result = a / b;
cout << "10 / 3.0 = " << result << endl; // 输出: 3.33333

// 两个整数相除，结果仍是整数
int c = 10, d = 3;
int intResult = c / d;
cout << "10 / 3 = " << intResult << endl; // 输出: 3

// 将结果赋给double变量，但计算时仍是整数除法
double doubleResult = c / d;
cout << "10 / 3 (赋给double) = " << doubleResult << endl; // 输出: 3

强制类型转换

程序员可以使用强制类型转换来显式指定数据类型：

int a = 10, b = 3;

// 强制将整数转换为double进行除法运算
double result = (double)a / b;
cout << "(double)10 / 3 = " << result << endl; // 输出: 3.33333

// 另一种写法
result = double(a) / b;
cout << "double(10) / 3 = " << result << endl; // 输出: 3.33333

表达式求值的注意事项

在使用表达式时需要注意一些特殊情况和潜在问题：

运算符优先级陷阱

复杂的表达式容易因为优先级问题产生意料之外的结果：

int a = 5, b = 3, c = 2;

// 可能产生混淆的表达式
int result1 = a + b * c; // 11，不是16
int result2 = (a + b) * c; // 16，使用括号明确意图
cout << "a + b * c = " << result1 << endl;
cout << "(a + b) * c = " << result2 << endl;

自增自减运算符的使用注意事项

在复杂表达式中使用自增自减运算符可能导致难以理解的代码：

int a = 5;
int result = a++ + ++a; // 结果依赖于编译器实现，应避免
cout << "result = " << result << endl;
cout << "a = " << a << endl;

// 更清晰的写法
int b = 5;
b++; // b = 6
++b; // b = 7
int result2 = b + b; // result2 = 14
cout << "清晰写法的结果 result2 = " << result2 << endl;

浮点数比较问题

由于浮点数的精度问题，直接使用关系运算符比较浮点数可能得到意外结果：

double x = 0.1 + 0.2;
double y = 0.3;

// 可能输出0(false)，尽管数学上0.1+0.2=0.3
cout << "x == y: " << (x == y) << endl;
cout << "x = " << x << endl;
cout << "y = " << y << endl;

// 更好的比较方式：使用误差范围
double epsilon = 1e-10;
bool equal = (x - y < epsilon) && (y - x < epsilon);
cout << "使用误差范围比较: " << equal << endl;

运算符的组合使用

多个运算符可以组合使用创建复杂的表达式，但需要注意可读性和正确性：

运算符组合的最佳实践

  • 使用括号：使用括号明确表达式的计算顺序
  • 避免过度复杂：过于复杂的表达式难以理解和维护
  • 保持一致性：在项目中保持运算符使用的风格一致

良好与不良的表达式示例

int a = 10, b = 5, c = 3, d = 2;

// 不推荐：难以理解的复杂表达式
int result1 = a + b * c - d++ + --c * b / a;

// 推荐：分步计算，清晰易懂
int temp1 = b * c; // 5 * 3 = 15
int temp2 = c - 1; // 3 - 1 = 2
int temp3 = temp2 * b; // 2 * 5 = 10
int temp4 = temp3 / a; // 10 / 10 = 1
int temp5 = d; // 2
d = d + 1; // d = 3
c = temp2; // c = 2
int result2 = a + temp1 - temp5 + temp4;
// 10 + 15 - 2 + 1 = 24

cout << "复杂表达式结果: " << result1 << endl;
cout << "分步计算结果: " << result2 << endl;

表达式优化建议

编写表达式时应考虑以下几点：

  • 优先保证代码的可读性，其次考虑性能
  • 复杂的表达式应该添加注释说明
  • 避免在表达式中修改多个变量的值
  • 使用有意义的变量名提高表达式的可理解性

实践应用

示例1：数学表达式计算

使用运算符计算数学表达式：

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 5, c = 3;
    int result;

    // 计算 (a + b) * c
    result = (a + b) * c;
    cout << "(a + b) * c = " << result << endl;

    // 计算 a² - b²
    result = a * a - b * b;
    cout << "a² - b² = " << result << endl;

    // 计算 (a + b)²
    result = (a + b) * (a + b);
    cout << "(a + b)² = " << result << endl;

    return 0;
}

示例2：使用自增/自减运算符

演示前置和后置自增/自减运算符的区别：

#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 5;
    int result1, result2;

    // 前置自增
    result1 = ++a;
    cout << "前置自增: ++a = " << result1 << ", a = " << a << endl;

    // 后置自增
    result2 = b++;
    cout << "后置自增: b++ = " << result2 << ", b = " << b << endl;

    return 0;
}

示例3：复合赋值运算符应用

使用复合赋值运算符简化表达式：

#include <iostream>
using namespace std;

int main() {
    int balance = 1000;
    int expense1 = 200, expense2 = 150, expense3 = 300;

    // 使用复合赋值运算符计算余额
    balance -= expense1; // balance = balance - expense1
    cout << "购买商品1后余额: " << balance << endl;

    balance -= expense2;
    cout << "购买商品2后余额: " << balance << endl;

    balance -= expense3;
    cout << "购买商品3后余额: " << balance << endl;

    // 存入工资
    int salary = 5000;
    balance += salary;
    cout << "存入工资后余额: " << balance << endl;

    return 0;
}

示例4：关系运算符应用

使用关系运算符比较数值：

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;

    // 输出关系运算结果（1表示真，0表示假）
    cout << "a = " << a << ", b = " << b << endl;
    cout << "a == b : " << (a == b) << endl;
    cout << "a != b : " << (a != b) << endl;
    cout << "a > b : " << (a > b) << endl;
    cout << "a < b : " << (a < b) << endl;
    cout << "a >= b : " << (a >= b) << endl;
    cout << "a <= b : " << (a <= b) << endl;

    return 0;
}
