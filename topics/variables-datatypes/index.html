什么是变量？

定义

变量是计算机内存中的一块命名存储区域，用于存放可变的数据（如数字、文本），可通过变量名访问和修改其值。

变量命名规则

  • 只能由 字母、数字、下划线 (_) 组成
  • 必须以 字母 或 下划线 开头
  • 不能使用 C++关键字（如 int, cout, return）
  • 区分大小写（如 age 和 Age 是不同变量）

正确示例： age, _score, num1

错误示例： 1age（数字开头）, cout（关键字）, &name（含非法字符）

变量的定义与赋值

语法：数据类型 变量名 = 初始值;

int age = 13; // 定义整型变量age并赋值13
double price = 19.9; // 定义双精度浮点型变量price
char grade = 'A'; // 定义字符型变量grade

基本数据类型与运算

类型 关键字 说明 示例
整型 int 存储整数（如年龄、数量） int count = 10;
双精度浮点型 double 存储小数，精度高 double price = 9.99;
单精度浮点型 float 存储小数，精度较低 float rate = 0.5f;
字符型 char 存储单个字符 char grade = 'A';
布尔型 bool 存储真（true）或假（false） bool isPass = true;

基本运算法则

C++支持常见的数学运算：

  • +：加法
  • -：减法
  • *：乘法
  • /：除法（整数除法会截断小数）
  • %：取模（求余数）
int a = 5, b = 12, c = 8;
int result = (a + b) * c; // 计算 (5+12)*8 = 136
cout << "结果是：" << result << endl;

从键盘获取输入

cin 输入流

cin 是 C++ 中用于从键盘读取输入的对象，使用 >> 流提取运算符。

int age;
cout << "请输入你的年龄：";
cin >> age; // 程序暂停，等待用户输入并回车
cout << "你输入的年龄是：" << age << endl;

连续输入

可以一次读取多个变量的值，用空格或回车分隔。

int n, a, x;
cin >> n >> a >> x; // 输入：12 2 3

🎯 项目实战：应用与计算

项目1：计算天安门广场面积

已知广场长880米，宽500米，计算面积。

#include <iostream>
using namespace std;

int main() {
    int length = 880, width = 500;
    int area = length * width;
    cout << area << endl; // 输出：440000
    return 0;
}

项目2：米老鼠背糖果

总糖果n颗，一次背a颗，背了x次，求剩余糖果。

#include <iostream>
using namespace std;

int main() {
    int n, a, x;
    cin >> n >> a >> x;
    int remaining = n - a * x;
    cout << remaining << endl;
    return 0;
}

项目3：小明买书找零

小明有n元，书原价m元，打8折，求剩余钱数。

#include <iostream>
using namespace std;

int main() {
    double n, m;
    cin >> n >> m;
    double cost = m * 0.8;
    double change = n - cost;
    cout << change << endl;
    return 0;
}
