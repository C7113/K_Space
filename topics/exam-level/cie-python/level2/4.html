Python二级课程第四课

字符串操作

课程目标

掌握Python字符串的基本概念、特点和常用操作，能够熟练使用字符串方法进行数据处理和格式化输出。

一、字符串基础

1. 字符串的概念与特点

字符串是Python中用于表示文本数据的数据类型，由一系列字符组成。

  • 不可变序列：字符串创建后不能修改，只能创建新的字符串
  • 有序集合：字符按顺序排列，支持索引和切片
  • 支持转义字符：使用反斜杠表示特殊字符
  • 多种定义方式：单引号、双引号、三引号
# 字符串定义示例
# 使用单引号
str1 = 'Hello, World!'

# 使用双引号
str2 = "Python Programming"

# 使用三引号（多行字符串）
str3 = '''这是一个
多行
字符串'''

# 转义字符
str4 = "这是第一行\n这是第二行" # \n 表示换行
str5 = "这是一个\"引号\"" # \" 表示双引号
str6 = '这是一个\'引号\'' # \' 表示单引号

print(str4)
# 输出：
# 这是第一行
# 这是第二行

2. 字符串比较

Python使用ASCII码（或Unicode）值进行字符串比较。

# 字符串比较示例
# 根据ASCII码比较
print('a' > 'A') # 输出：True（a的ASCII码是97，A是65）
print('abc' > 'aaaaa') # 输出：True（先比较第一个字符，相同再比较下一个）

# 字符串比较规则：
# 1. 比较第一个字符的ASCII码
# 2. 如果相同，比较下一个字符
# 3. 直到找到不同的字符或一个字符串结束

print('apple' < 'banana') # 输出：True（a的ASCII码小于b）
print('cat' < 'category') # 输出：True（前三个字符相同，但第一个字符串较短）
ASCII码小知识： 'A'=65, 'a'=97, '0'=48。小写字母 > 大写字母 > 数字。

二、字符串常用操作

1. 字符串查询与统计

方法 说明 示例
len(str) 返回字符串长度 len("hello") → 5
str.count(sub) 返回子串sub在字符串中出现的次数 "hello".count("l") → 2
str.find(sub) 返回子串sub第一次出现的索引，不存在返回-1 "hello".find("l") → 2
str.rfind(sub) 返回子串sub最后一次出现的索引，不存在返回-1 "hello".rfind("l") → 3
str.index(sub) 返回子串sub第一次出现的索引，不存在抛出错误 "hello".index("l") → 2
str.rindex(sub) 返回子串sub最后一次出现的索引，不存在抛出错误 "hello".rindex("l") → 3
# 字符串查询与统计示例
text = "Python programming is fun and Python is powerful"

# 获取字符串长度
length = len(text)
print("字符串长度:", length) # 输出：字符串长度: 45

# 统计子串出现次数
count_python = text.count("Python")
print("Python出现次数:", count_python) # 输出：Python出现次数: 2

# 查找子串位置
pos1 = text.find("Python")
print("Python第一次出现位置:", pos1) # 输出：Python第一次出现位置: 0

pos2 = text.rfind("Python")
print("Python最后一次出现位置:", pos2) # 输出：Python最后一次出现位置: 24

# 使用index查找（找不到会报错）
pos3 = text.index("is")
print("is第一次出现位置:", pos3) # 输出：is第一次出现位置: 17

# find和index的区别
not_found_find = text.find("Java") # 返回-1
# not_found_index = text.index("Java") # 会报错：ValueError

2. 字符串大小写转换

方法 说明 示例
str.upper() 将字符串转换为大写 "hello".upper() → "HELLO"
str.lower() 将字符串转换为小写 "HELLO".lower() → "hello"
str.capitalize() 将字符串首字母大写，其余小写 "hello WORLD".capitalize() → "Hello world"
# 字符串大小写转换示例
text = "Python Programming"

# 转换为大写
upper_text = text.upper()
print("大写:", upper_text) # 输出：大写: PYTHON PROGRAMMING

# 转换为小写
lower_text = text.lower()
print("小写:", lower_text) # 输出：小写: python programming

# 首字母大写
capitalized = "hello world".capitalize()
print("首字母大写:", capitalized) # 输出：首字母大写: Hello world

# 实际应用：不区分大小写的比较
user_input = "Yes"
if user_input.lower() == "yes":
    print("用户同意了")

3. 字符串判断方法

方法 说明 示例
str.isalpha() 判断字符串是否只包含字母 "Hello".isalpha() → True
str.isdigit() 判断字符串是否只包含数字 "123".isdigit() → True
str.isalnum() 判断字符串是否只包含字母和数字 "Hello123".isalnum() → True
str.isspace() 判断字符串是否只包含空白字符 " \t\n".isspace() → True
str.startswith(prefix) 判断字符串是否以指定前缀开头 "hello".startswith("he") → True
str.endswith(suffix) 判断字符串是否以指定后缀结尾 "hello".endswith("lo") → True
# 字符串判断示例
# 检查是否只包含字母
print("Hello".isalpha()) # 输出：True
print("Hello123".isalpha()) # 输出：False

# 检查是否只包含数字
print("123".isdigit()) # 输出：True
print("12.3".isdigit()) # 输出：False（小数点不是数字）

# 检查是否只包含字母和数字
print("Hello123".isalnum()) # 输出：True
print("Hello 123".isalnum()) # 输出：False（空格不是字母或数字）

# 检查是否只包含空白字符
print(" \t\n".isspace()) # 输出：True
print("hello".isspace()) # 输出：False

# 检查开头和结尾
filename = "document.txt"
print(filename.startswith("doc")) # 输出：True
print(filename.endswith(".txt")) # 输出：True

4. 字符串去除空白

方法 说明 示例
str.strip() 去除字符串开头和结尾的空白字符 " hello ".strip() → "hello"
str.lstrip() 去除字符串开头的空白字符 " hello ".lstrip() → "hello "
str.rstrip() 去除字符串结尾的空白字符 " hello ".rstrip() → " hello"
# 字符串去除空白示例
text = " Hello, World! "

# 去除两端空白
stripped = text.strip()
print("去除两端空白:", f"'{stripped}'") # 输出：去除两端空白: 'Hello, World!'

# 去除左端空白
left_stripped = text.lstrip()
print("去除左端空白:", f"'{left_stripped}'") # 输出：去除左端空白: 'Hello, World! '

# 去除右端空白
right_stripped = text.rstrip()
print("去除右端空白:", f"'{right_stripped}'") # 输出：去除右端空白: ' Hello, World!'

# 实际应用：用户输入处理
user_input = " yes "
if user_input.strip().lower() == "yes":
    print("去除空白后匹配成功")

5. 字符串分割与连接

方法 说明 示例
str.split(sep) 使用分隔符sep分割字符串，返回列表 "a,b,c".split(",") → ["a","b","c"]
str.join(iterable) 使用字符串连接可迭代对象中的元素 ",".join(["a","b","c"]) → "a,b,c"
# 字符串分割与连接示例
# 分割字符串
csv_data = "apple,banana,orange,grape"
fruits = csv_data.split(",")
print("分割结果:", fruits) # 输出：分割结果: ['apple', 'banana', 'orange', 'grape']

# 限制分割次数
limited_split = csv_data.split(",", 2)
print("限制分割次数:", limited_split) # 输出：限制分割次数: ['apple', 'banana', 'orange,grape']

# 连接字符串
fruit_list = ["apple", "banana", "orange"]
joined = ", ".join(fruit_list)
print("连接结果:", joined) # 输出：连接结果: apple, banana, orange

# 使用不同连接符
path = "/".join(["home", "user", "documents"])
print("路径:", path) # 输出：路径: home/user/documents

# 实际应用：处理多行文本
multi_line = "第一行\n第二行\n第三行"
lines = multi_line.split("\n")
print("行列表:", lines) # 输出：行列表: ['第一行', '第二行', '第三行']

6. 字符串替换

方法 说明 示例
str.replace(old, new) 将字符串中的old替换为new "hello".replace("l", "x") → "hexxo"
str.replace(old, new, count) 将字符串中的old替换为new，最多替换count次 "hello".replace("l", "x", 1) → "hexlo"
# 字符串替换示例
text = "I like apples and apples are tasty"

# 替换所有匹配项
replaced_all = text.replace("apples", "oranges")
print("全部替换:", replaced_all) # 输出：全部替换: I like oranges and oranges are tasty

# 替换指定次数
replaced_once = text.replace("apples", "oranges", 1)
print("替换一次:", replaced_once) # 输出：替换一次: I like oranges and apples are tasty

# 实际应用：数据清洗
dirty_text = " Hello, World! "
cleaned = dirty_text.replace(" ", " ").strip()
print("清洗后:", f"'{cleaned}'") # 输出：清洗后: 'Hello, World!'

三、字符串格式化

1. format() 方法

format() 方法用于格式化字符串，可以按位置或关键字插入值。

# format() 方法示例
# 按位置格式化
text1 = "{} {}".format("Hello", "World")
print(text1) # 输出：Hello World

# 按索引格式化
text2 = "{1} {0}".format("World", "Hello")
print(text2) # 输出：Hello World

# 按关键字格式化
text3 = "{name} is {age} years old".format(name="Alice", age=25)
print(text3) # 输出：Alice is 25 years old

# 格式化数字
text4 = "Value: {:.2f}".format(3.14159)
print(text4) # 输出：Value: 3.14

# 格式化百分比
text5 = "Success rate: {:.1%}".format(0.856)
print(text5) # 输出：Success rate: 85.6%

2. f-string 格式化（Python 3.6+）

f-string 是一种更简洁、更直观的字符串格式化方法。

# f-string 示例
name = "Alice"
age = 25

# 基本用法
text1 = f"{name} is {age} years old"
print(text1) # 输出：Alice is 25 years old

# 表达式计算
text2 = f"Next year, {name} will be {age + 1} years old"
print(text2) # 输出：Next year, Alice will be 26 years old

# 格式化数字
pi = 3.14159
text3 = f"Pi value: {pi:.2f}"
print(text3) # 输出：Pi value: 3.14

# 调用方法
text4 = f"Name in uppercase: {name.upper()}"
print(text4) # 输出：Name in uppercase: ALICE

3. % 格式化（传统方法）

% 操作符是Python早期的字符串格式化方法，现在仍被广泛使用。

# % 格式化示例
name = "Bob"
age = 30

# 基本用法
text1 = "%s is %d years old" % (name, age)
print(text1) # 输出：Bob is 30 years old

# 格式化浮点数
price = 19.99
text2 = "Price: $%.2f" % price
print(text2) # 输出：Price: $19.99

# 格式化百分比
ratio = 0.75
text3 = "Completion: %.1f%%" % (ratio * 100)
print(text3) # 输出：Completion: 75.0%
格式化符号： %s=字符串, %d=整数, %f=浮点数, %%=百分号。

四、字符串内置函数

常用内置函数

函数 说明 示例

五、例题训练

题目1：字符串基本操作

给定字符串：text = " Python Programming is FUN! "

完成以下操作：

  1. 去除字符串两端的空白字符
  2. 将字符串转换为全小写
  3. 统计字符串中字母"m"出现的次数
  4. 将"FUN"替换为"powerful"
  5. 将字符串按空格分割成单词列表
# 答案
text = " Python Programming is FUN! "
print("原字符串:", f"'{text}'")

# 1. 去除字符串两端的空白字符
trimmed = text.strip()
print("去除空白后:", f"'{trimmed}'")

# 2. 将字符串转换为全小写
lower_text = trimmed.lower()
print("转换为小写:", lower_text)

# 3. 统计字符串中字母"m"出现的次数
count_m = trimmed.count('m')
print("m出现次数:", count_m)

# 4. 将"FUN"替换为"powerful"
replaced = trimmed.replace("FUN", "powerful")
print("替换后:", replaced)

# 5. 将字符串按空格分割成单词列表
words = trimmed.split()
print("单词列表:", words)

题目2：字符串格式化

创建一个学生信息字典：student = {"name": "张三", "age": 18, "score": 92.5}

使用三种不同的字符串格式化方法输出以下信息：

  1. 学生张三今年18岁，考试成绩92.5分
  2. 学生张三的考试成绩是92.50分（保留两位小数）
  3. 学生张三的考试优秀率是92.5%（添加百分号）
# 答案
student = {"name": "张三", "age": 18, "score": 92.5}

# 1. 使用format方法
text1 = "学生{}今年{}岁，考试成绩{}分".format(student["name"], student["age"], student["score"])
print("format方法:", text1)

# 2. 使用f-string方法
text2 = f"学生{student['name']}今年{student['age']}岁，考试成绩{student['score']}分"
print("f-string方法:", text2)

# 3. 使用%方法
text3 = "学生%s今年%d岁，考试成绩%.1f分" % (student["name"], student["age"], student["score"])
print("%方法:", text3)

# 保留两位小数
text4 = f"学生{student['name']}的考试成绩是{student['score']:.2f}分"
print("保留两位小数:", text4)

# 添加百分号
text5 = f"学生{student['name']}的考试优秀率是{student['score']:.1f}%"
print("添加百分号:", text5)

题目3：字符串处理综合应用

给定一个包含多个邮箱地址的字符串：emails = "user1@example.com; user2@test.org; user3@demo.net"

完成以下操作：

  1. 将字符串按分号分割成单个邮箱地址
  2. 去除每个邮箱地址两端的空白字符
  3. 提取每个邮箱的域名（@后面的部分）
  4. 统计不同域名的出现次数
  5. 将所有邮箱地址用逗号连接成一个字符串
# 答案
emails = "user1@example.com; user2@test.org; user3@demo.net"
print("原字符串:", emails)

# 1. 将字符串按分号分割成单个邮箱地址
email_list = emails.split(";")
print("分割后的邮箱列表:", email_list)

# 2. 去除每个邮箱地址两端的空白字符
cleaned_emails = [email.strip() for email in email_list]
print("去除空白后的邮箱列表:", cleaned_emails)

# 3. 提取每个邮箱的域名（@后面的部分）
domains = [email.split('@')[1] for email in cleaned_emails]
print("域名列表:", domains)

# 4. 统计不同域名的出现次数
from collections import Counter
domain_count = Counter(domains)
print("域名出现次数:", domain_count)

# 5. 将所有邮箱地址用逗号连接成一个字符串
joined_emails = ", ".join(cleaned_emails)
print("连接后的字符串:", joined_emails)
