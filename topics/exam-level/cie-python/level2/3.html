元组与集合

课程目标

掌握Python元组和集合的基本概念、特点和常用操作，理解它们与列表的区别及应用场景。

一、元组（Tuple）

1. 元组的概念与特点

元组是Python中的不可变序列，用于存储一组有序的元素。

  • 不可变序列：一旦创建，元素不能修改
  • 有序集合：元素按添加顺序排列
  • 固定长度：创建后长度不可变
  • 可任意嵌套：可以包含其他元组或列表
  • 异质性：可以包含不同类型的元素
# 元组的特点示例
# 元组可以包含不同类型的数据
mixed_tuple = (1, "hello", 3.14, True)
print(mixed_tuple) # 输出：(1, "hello", 3.14, True)

# 元组可以嵌套
nested_tuple = (1, (2, 3), [4, 5])
print(nested_tuple) # 输出：(1, (2, 3), [4, 5])

# 元组不可变
my_tuple = (1, 2, 3)
# my_tuple[0] = 10 # 这行会报错：TypeError

2. 元组的定义方法

定义方法 示例 说明
使用小括号 tuple1 = (1, 2, 3, 4, 5) 标准定义方式
省略小括号 tuple1 = 1, 2, 3, 4, 5 Python允许省略小括号
单个元素元组 tuple1 = (1,) 必须加逗号，否则是整数类型
空元组 tuple1 = () 空元组的定义
使用tuple()函数 tuple1 = tuple([1, 2, 3]) 将其他序列转换为元组
# 元组定义示例
# 标准定义
tuple1 = (1, 2, 3, 4, 5)
print(tuple1) # 输出：(1, 2, 3, 4, 5)

# 省略小括号
tuple2 = 1, 2, 3, 4, 5
print(tuple2) # 输出：(1, 2, 3, 4, 5)

# 单个元素元组（必须加逗号）
single_tuple = (1,)
print(single_tuple) # 输出：(1,)
print(type(single_tuple)) # 输出：<class 'tuple'>

# 错误示例：没有逗号是整数
not_tuple = (1)
print(type(not_tuple)) # 输出：<class 'int'>

# 空元组
empty_tuple = ()
print(empty_tuple) # 输出：()

# 使用tuple()函数转换
list_to_tuple = tuple([1, 2, 3])
print(list_to_tuple) # 输出：(1, 2, 3)
重要：定义单个元素的元组时，必须在元素后加逗号，否则Python会将其视为普通变量而不是元组。

3. 元组的常用操作

由于元组是不可变序列，所以只有查询操作，没有增删改操作。

操作 说明 示例
元组[索引] 根据索引取值 tuple1[0]
元组.index(元素) 返回元素第一次出现的索引 tuple1.index(2)
元组.count(元素) 返回元素在元组中出现的次数 tuple1.count(2)
len(元组) 返回元组长度（元素个数） len(tuple1)
元素 in 元组 检查元素是否在元组中 2 in tuple1
元组切片 获取元组的一部分 tuple1[1:3]
# 元组操作示例
my_tuple = (10, 20, 30, 20, 40, 50)

# 索引访问
print(my_tuple[0]) # 输出：10
print(my_tuple[-1]) # 输出：50

# 查找元素索引
index = my_tuple.index(30)
print("30的索引:", index) # 输出：30的索引: 2

# 统计元素出现次数
count = my_tuple.count(20)
print("20出现的次数:", count) # 输出：20出现的次数: 2

# 获取元组长度
length = len(my_tuple)
print("元组长度:", length) # 输出：元组长度: 6

# 检查元素是否存在
has_30 = 30 in my_tuple
print("是否有30:", has_30) # 输出：是否有30: True

# 元组切片
slice_result = my_tuple[1:4]
print("切片结果:", slice_result) # 输出：切片结果: (20, 30, 20)

4. 元组的拼接与复制

元组支持使用 + 进行拼接和 * 进行复制。

# 元组拼接与复制示例
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)

# 元组拼接
combined = tuple1 + tuple2
print("拼接结果:", combined) # 输出：拼接结果: (1, 2, 3, 4, 5, 6)

# 元组复制
repeated = tuple1 * 3
print("复制结果:", repeated) # 输出：复制结果: (1, 2, 3, 1, 2, 3, 1, 2, 3)

二、集合（Set）

1. 集合的概念与特点

集合是Python中的无序、不重复元素序列。

  • 无序性：元素没有固定顺序，不能通过索引访问
  • 不重复性：自动去除重复元素
  • 可变性：可以添加、删除元素
  • 数学运算：支持交集、并集、差集等数学运算
# 集合的特点示例
# 集合自动去重
my_set = {1, 2, 2, 3, 3, 3, 4}
print(my_set) # 输出：{1, 2, 3, 4}

# 集合无序，每次输出顺序可能不同
unordered_set = {"apple", "banana", "orange"}
print(unordered_set) # 输出可能是：{"banana", "apple", "orange"}

# 不能通过索引访问
# print(my_set[0]) # 这行会报错：TypeError

2. 集合的定义方法

定义方法 示例 说明
使用花括号 set1 = {1, 2, 3, 4, 5} 标准定义方式
使用set()函数 set1 = set([1, 2, 3, 4, 5]) 将其他序列转换为集合
空集合 set1 = set() 不能使用{}，那是空字典
# 集合定义示例
# 使用花括号定义
set1 = {1, 2, 3, 4, 5}
print(set1) # 输出：{1, 2, 3, 4, 5}

# 使用set()函数转换
list_to_set = set([1, 2, 2, 3, 3, 3])
print(list_to_set) # 输出：{1, 2, 3}

# 空集合（注意：不能用{}）
empty_set = set()
print(empty_set) # 输出：set()
print(type(empty_set)) # 输出：<class 'set'>

# 错误示例：{}创建的是字典
not_set = {}
print(type(not_set)) # 输出：<class 'dict'>
注意：创建空集合必须使用set()函数，不能使用{}，因为{}创建的是空字典。

3. 集合的常用操作

操作 说明 示例
集合.add(元素) 添加元素到集合 set1.add(6)
集合.remove(元素) 删除指定元素，元素不存在会报错 set1.remove(2)
集合.discard(元素) 删除指定元素，元素不存在不会报错 set1.discard(2)
集合.pop() 随机删除并返回一个元素 set1.pop()
集合.clear() 清空集合 set1.clear()
len(集合) 返回集合元素个数 len(set1)
元素 in 集合 检查元素是否在集合中 2 in set1
# 集合操作示例
my_set = {1, 2, 3, 4, 5}

# 添加元素
my_set.add(6)
print("添加6后:", my_set) # 输出：添加6后: {1, 2, 3, 4, 5, 6}

# 删除元素（remove）
my_set.remove(2)
print("删除2后:", my_set) # 输出：删除2后: {1, 3, 4, 5, 6}

# 删除元素（discard）
my_set.discard(10) # 10不存在，但不会报错
print("discard 10后:", my_set) # 输出：discard 10后: {1, 3, 4, 5, 6}

# 随机删除一个元素
removed = my_set.pop()
print("删除的元素:", removed)
print("删除后:", my_set)

# 检查元素是否存在
has_3 = 3 in my_set
print("是否有3:", has_3) # 输出：是否有3: True

# 获取集合长度
length = len(my_set)
print("集合长度:", length)

4. 集合的数学运算

运算 说明 示例
集合1 & 集合2 交集：返回两个集合共有的元素 set1 & set2
集合1 | 集合2 并集：返回两个集合所有的元素 set1 | set2
集合1 - 集合2 差集：返回在集合1但不在集合2的元素 set1 - set2
集合1 ^ 集合2 对称差集：返回不同时属于两个集合的元素 set1 ^ set2
# 集合运算示例
setA = {1, 2, 3, 4, 5}
setB = {4, 5, 6, 7, 8}

# 交集
intersection = setA & setB
print("交集:", intersection) # 输出：交集: {4, 5}

# 并集
union = setA | setB
print("并集:", union) # 输出：并集: {1, 2, 3, 4, 5, 6, 7, 8}

# 差集
difference = setA - setB
print("A-B差集:", difference) # 输出：A-B差集: {1, 2, 3}

# 对称差集
symmetric_diff = setA ^ setB
print("对称差集:", symmetric_diff) # 输出：对称差集: {1, 2, 3, 6, 7, 8}

三、列表、元组与集合的比较

特性 列表(List) 元组(Tuple) 集合(Set)
可变性 可变 不可变 可变
有序性 有序 有序 无序
重复元素 允许 允许 不允许
索引访问 支持 支持 不支持
定义符号 [] () {}
空对象 [] () set()
应用场景 需要修改的数据集合 不需要修改的数据集合 去重、成员测试、数学运算

题目1：元组基本操作

创建一个包含5个元素的元组，然后：

  1. 打印第二个和最后一个元素
  2. 查找元素30的位置
  3. 统计元素20出现的次数
  4. 检查元素50是否在元组中
  5. 将元组与另一个元组(60,70)拼接
# 答案
# 创建元组
my_tuple = (10, 20, 30, 20, 40)
print("原元组:", my_tuple)

# 1. 打印第二个和最后一个元素
print("第二个元素:", my_tuple[1]) # 输出：20
print("最后一个元素:", my_tuple[-1]) # 输出：40

# 2. 查找元素30的位置
pos = my_tuple.index(30)
print("30的位置:", pos) # 输出：2

# 3. 统计元素20出现的次数
count = my_tuple.count(20)
print("20出现的次数:", count) # 输出：2

# 4. 检查元素50是否在元组中
has_50 = 50 in my_tuple
print("是否有50:", has_50) # 输出：False

# 5. 与另一个元组拼接
new_tuple = my_tuple + (60, 70)
print("拼接后:", new_tuple) # 输出：(10, 20, 30, 20, 40, 60, 70)

题目2：集合基本操作

创建两个集合：setA = {1, 2, 3, 4, 5} 和 setB = {4, 5, 6, 7, 8}

完成以下操作：

  1. 向setA添加元素10
  2. 从setB删除元素8
  3. 计算两个集合的交集
  4. 计算两个集合的并集
  5. 计算setA中有但setB中没有的元素
# 答案
setA = {1, 2, 3, 4, 5}
setB = {4, 5, 6, 7, 8}
print("setA:", setA)
print("setB:", setB)

# 1. 向setA添加元素10
setA.add(10)
print("添加10后setA:", setA)

# 2. 从setB删除元素8
setB.remove(8)
print("删除8后setB:", setB)

# 3. 计算两个集合的交集
intersection = setA & setB
print("交集:", intersection) # 输出：{4, 5}

# 4. 计算两个集合的并集
union = setA | setB
print("并集:", union) # 输出：{1, 2, 3, 4, 5, 6, 7, 10}

# 5. 计算setA中有但setB中没有的元素
difference = setA - setB
print("差集:", difference) # 输出：{1, 2, 3, 10}

题目3：数据去重与转换

给定一个包含重复元素的列表：[1, 2, 2, 3, 4, 4, 4, 5, 5]

完成以下操作：

  1. 使用集合去除重复元素
  2. 将去重后的集合转换回列表
  3. 将列表转换为元组
  4. 创建一个空集合和一个空字典，验证它们的类型
# 答案
original_list = [1, 2, 2, 3, 4, 4, 4, 5, 5]
print("原列表:", original_list)

# 1. 使用集合去除重复元素
unique_set = set(original_list)
print("去重后的集合:", unique_set) # 输出：{1, 2, 3, 4, 5}

# 2. 将去重后的集合转换回列表
unique_list = list(unique_set)
print("去重后的列表:", unique_list) # 输出：[1, 2, 3, 4, 5]

# 3. 将列表转换为元组
unique_tuple = tuple(unique_list)
print("转换后的元组:", unique_tuple) # 输出：(1, 2, 3, 4, 5)

# 4. 创建空集合和空字典
empty_set = set()
empty_dict = {}
print("空集合类型:", type(empty_set)) # 输出：<class 'set'>
print("空字典类型:", type(empty_dict)) # 输出：<class 'dict'>

课程要点

元组核心概念

  • 不可变序列
  • 有序集合
  • 固定长度
  • 可任意嵌套

元组注意事项

  • 单个元素元组必须加逗号
  • 只有查询操作，没有增删改
  • 支持索引和切片
  • 支持拼接和复制

集合核心概念

  • 无序、不重复
  • 自动去重
  • 不支持索引
  • 支持数学运算

集合注意事项

  • 空集合必须用set()
  • {}创建的是空字典
  • remove()元素不存在会报错
  • discard()元素不存在不会报错

常见错误

  • 忘记单个元素元组的逗号
  • 尝试修改元组元素
  • 用{}创建空集合
  • 对集合使用索引
