知识点详解

1. 程序基本逻辑模型

任何程序都可以抽象为三个基本步骤：输入 → 处理 → 输出

输入（Input）

获取外部数据，通常来自用户的键盘输入或其他数据源。

处理（Process）

对输入的数据进行计算、变换或逻辑判断等操作。

输出（Output）

将处理结果展示给用户，通常是显示在屏幕上或保存到文件中。

# 程序基本结构示例 # 输入：获取用户姓名和年龄 name = input("请输入您的姓名：") age = input("请输入您的年龄：") # 处理：计算出生年份 current_year = 2025 birth_year = current_year - int(age) # 输出：显示结果 print("您好，", name) print("您出生于", birth_year, "年")

2. input() 函数详解

input() 函数用于从用户那里获取输入数据。

基本语法

variable = input([prompt])

参数说明：

  • prompt（可选）：提示信息，显示给用户看
  • 返回值：用户输入的内容（始终是字符串类型）

使用示例

# 不带提示信息的输入 data = input() print("您输入的是：", data) # 带提示信息的输入 name = input("请输入您的姓名：") print("您好，", name)

重要提醒

input() 函数返回的始终是字符串类型，即使输入的是数字也需要转换：

# 错误示例 age = input("请输入您的年龄：") # 用户输入 18 result = age + 5 # 结果是 "185" 而不是 23 # 正确示例 age = input("请输入您的年龄：") # 用户输入 18 age = int(age) # 转换为整数 result = age + 5 # 结果是 23

3. print() 函数详解

print() 函数用于向屏幕输出信息。

基本语法

print(value, ..., sep=' ', end='\n', file=sys.stdout, flush=False)

常用参数：

  • value,...：要输出的一个或多个值
  • sep：分隔符，默认为空格
  • end：结束符，默认为换行符

使用示例

# 输出单个值
print("Hello World!")

# 输出多个值（默认用空格分隔）
print("姓名：", "张三")

# 自定义分隔符
print("苹果", "香蕉", "橙子", sep=", ")

# 自定义结束符（不换行）
print("第一部分", end="")
print("第二部分")
# 输出：第一部分第二部分

4. 良好代码习惯

合理使用注释

使用 # 符号添加单行注释，解释代码的作用：

# 计算矩形面积
length = 10  # 长度
width = 5    # 宽度
area = length * width  # 面积 = 长 × 宽
print("矩形面积为：", area)

正确使用缩进

Python 使用缩进来表示代码块，推荐使用 4 个空格：

# 正确的缩进
if True:
    print("这是正确的缩进")
    print("同一代码块中的语句")

# 错误的缩进（混用空格和制表符）
if True:
    print("使用空格")
	print("使用制表符")  # 这里会出错

字符串引号规范

字符串必须用英文引号包围，可以使用单引号或双引号：

# 合法的字符串表示
message1 = "Hello World"
message2 = 'Hello World'
message3 = "他说：'你好'"
message4 = '她说："再见"'

# 非法的字符串表示（中文引号）
message5 = "你好"  # 错误：使用了中文引号

示例2：简单计算器程序

# calculator.py
# 简单的加法计算器

# 输入阶段
print("=== 简单加法计算器 ===")
num1 = input("请输入第一个数字：")
num2 = input("请输入第二个数字：")

# 处理阶段
# 注意：需要将字符串转换为数字
result = int(num1) + int(num2)

# 输出阶段
print("计算结果：", num1, "+", num2, "=", result)

运行结果：

=== 简单加法计算器 ===
请输入第一个数字：25
请输入第二个数字：17
计算结果： 25 + 17 = 42
