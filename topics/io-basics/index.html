输出操作

cout对象

cout是C++中用于标准输出的对象，通常将数据显示到控制台屏幕上。

基本输出语法

使用<<流插入运算符将数据发送到输出流：

cout << 数据;
cout << 数据1 << 数据2 << 数据3;

输出操作的特点

  • 连续输出：可以连续使用多个<<运算符输出多个数据
  • 自动类型识别：cout能自动识别不同数据类型的输出格式
  • 链式操作：<<运算符返回cout对象本身，支持链式操作

基本输出示例

#include <iostream>
using namespace std;

int main() {
    int age = 15;
    double height = 1.75;
    char grade = 'A';
    string name = "小明";

    // 基本输出
    cout << "Hello World!" << endl;

    // 输出多个数据
    cout << "姓名: " << name << endl;
    cout << "年龄: " << age << "岁" << endl;
    cout << "身高: " << height << "米" << endl;
    cout << "成绩: " << grade << "等" << endl;

    // 链式输出
    cout << "姓名: " << name << ", 年龄: " << age << "岁" << endl;

    return 0;
}

endl与换行符

在输出时，我们经常需要换行操作：

endl操作符

  • 功能：输出换行符并刷新输出缓冲区
  • 使用：cout << "内容" << endl;

换行符'\n'

  • 功能：仅输出换行符，不刷新缓冲区
  • 使用：cout << "内容\n";

换行操作示例

#include <iostream>
using namespace std;

int main() {
    // 使用endl换行
    cout << "第一行" << endl;
    cout << "第二行" << endl;

    // 使用\n换行
    cout << "第三行\n";
    cout << "第四行\n";

    // 连续换行
    cout << "第五行" << endl << endl; // 输出一行空行
    cout << "第六行\n\n"; // 输出一行空行

    return 0;
}

输入操作

cin对象

cin是C++中用于标准输入的对象，通常从键盘读取用户输入的数据。

基本输入语法

使用>>流提取运算符从输入流中读取数据：

cin >> 变量;
cin >> 变量1 >> 变量2 >> 变量3;

输入操作的特点

  • 连续输入：可以连续使用多个>>运算符读取多个数据
  • 空白符分隔：输入数据之间需要用空格、制表符或回车符分隔
  • 类型安全：cin会根据变量类型自动解析输入数据

基本输入示例

#include <iostream>
using namespace std;

int main() {
    int age;
    double height;
    char grade;
    string name;

    // 提示用户输入
    cout << "请输入您的年龄: ";
    cin >> age;

    cout << "请输入您的身高: ";
    cin >> height;

    cout << "请输入您的成绩等级: ";
    cin >> grade;

    // 输出输入的结果
    cout << "您输入的信息如下:" << endl;
    cout << "年龄: " << age << "岁" << endl;
    cout << "身高: " << height << "米" << endl;
    cout << "成绩: " << grade << "等" << endl;

    return 0;
}

getline函数

对于字符串输入，特别是包含空格的字符串，可以使用getline函数：

getline语法

getline(cin, 字符串变量);
getline(cin, 字符串变量, 分隔符);

getline的特点

  • 整行读取：可以读取包含空格的整行文本
  • 自定义分隔符：可以指定除换行符外的其他字符作为分隔符
  • 适用类型：主要用于string类型

getline使用示例

#include <iostream>
#include <string>
using namespace std;

int main() {
    string fullName;
    string sentence;

    cout << "请输入您的全名: ";
    getline(cin, fullName); // 可以读取包含空格的姓名

    cout << "请输入一句话: ";
    getline(cin, sentence);

    cout << "您的全名是: " << fullName << endl;
    cout << "您输入的话是: " << sentence << endl;

    return 0;
}

C风格输入输出函数

除了C++风格的cin/cout，C++也支持C语言的printf/scanf函数：

printf函数

printf函数用于格式化输出，需要包含<cstdio>头文件：

printf(格式字符串, 参数1, 参数2, ...);

常用格式说明符

格式说明符 对应数据类型 示例
%d 或 %i int（整数） printf("%d", 123);
%f float（单精度浮点数） printf("%f", 3.14f);
%lf double（双精度浮点数） printf("%lf", 3.14159);
%c char（字符） printf("%c", 'A');
%s char[]（字符串） printf("%s", "Hello");

printf使用示例

#include <iostream>
#include <cstdio>
using namespace std;

int main() {
    int age = 15;
    double height = 1.75;
    char grade = 'A';
    char name[] = "小明";

    // 基本格式化输出
    printf("姓名: %s\n", name);
    printf("年龄: %d岁\n", age);
    printf("身高: %.2lf米\n", height);
    printf("成绩: %c等\n", grade);

    // 格式控制
    printf("右对齐:%6d\n", 123);
    printf("左对齐:%-6d\n", 123);
    printf("补零:%06d\n", 123);
    printf("保留2位小数: %.2f\n", 3.14159);

    return 0;
}

scanf函数

scanf函数用于格式化输入，同样需要包含<cstdio>头文件：

scanf(格式字符串, 地址1, 地址2, ...);

scanf的特点

  • 需要传递变量的地址（使用&取地址符）
  • 按照格式说明符解析输入数据
  • 以空白字符（空格、制表符、换行符）作为分隔符

scanf使用示例

#include <iostream>
#include <cstdio>
using namespace std;

int main() {
    int age;
    double height;
    char grade;
    char name[50];

    // 提示用户输入
    printf("请输入姓名: ");
    scanf("%s", name); // 注意：字符串不需要&

    printf("请输入年龄: ");
    scanf("%d", &age); // 注意：需要使用&取地址

    printf("请输入身高: ");
    scanf("%lf", &height);

    printf("请输入成绩等级: ");
    scanf(" %c", &grade); // 注意：前面加空格跳过换行符

    // 输出结果
    printf("\n您输入的信息如下:\n");
    printf("姓名: %s\n", name);
    printf("年龄: %d岁\n", age);
    printf("身高: %.2lf米\n", height);
    printf("成绩: %c等\n", grade);

    return 0;
}

C风格与C++风格对比

两种输入输出方式各有优缺点：

C++风格（cin/cout）

  • 优点：类型安全、易于使用、支持重载、可扩展
  • 缺点：性能略低、格式化功能较弱

C风格（scanf/printf）

  • 优点：性能高、格式化功能强大、控制精确
  • 缺点：类型不安全、容易出错、不支持自定义类型

两种风格对比示例

#include <iostream>
#include <cstdio>
#include <iomanip>
using namespace std;

int main() {
    int number = 42;
    double pi = 3.14159;

    // C++风格输出
    cout << "C++风格: " << number << " "
            << fixed << setprecision(2) << pi << endl;

    // C风格输出
    printf("C风格: %d %.2lf\n", number, pi);

    return 0;
}

格式化输出

iomanip头文件

通过包含<iomanip>头文件，可以使用更多格式化输出功能：

常用格式化操作符

操作符 功能 示例
setw(n) 设置输出宽度为n个字符 cout << setw(10) << 25;
setfill(c) 设置填充字符为c cout << setfill('0') << setw(5) << 25;
setprecision(n) 设置浮点数精度为n位 cout << setprecision(3) << 3.14159;
fixed 固定小数点表示法 cout << fixed << setprecision(2) << 3.14159;
left/right 左对齐/右对齐 cout << left << setw(10) << "文本";

格式化输出示例

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int number = 42;
    double pi = 3.14159265;
    string text = "C++";

    // 设置宽度和填充字符
    cout << "默认: " << number << endl;
    cout << "宽度为10: " << setw(10) << number << endl;
    cout << "用0填充: " << setfill('0') << setw(10) << number << endl;
    cout << "恢复默认填充: " << setfill(' ') << setw(10) << number << endl;

    // 浮点数格式化
    cout << "默认pi: " << pi << endl;
    cout << "保留3位小数: " << setprecision(3) << pi << endl;
    cout << "固定小数点: " << fixed << setprecision(2) << pi << endl;

    // 对齐方式
    cout << "右对齐: " << right << setw(10) << text << endl;
    cout << "左对齐: " << left << setw(10) << text << endl;
    cout << "居中对齐: " << right << setw(7) << text << left << setw(3) << "" << endl;

    return 0;
}

格式化输入

输入也可以通过一些方式实现格式化：

跳过特定字符

在输入时可以跳过特定字符：

cin >> number1 >> ws >> number2; // 跳过空白字符
cin >> number1 >> "," >> number2; // 跳过逗号

格式化输入示例

#include <iostream>
using namespace std;

int main() {
    int a, b;
    char comma;

    cout << "请输入两个用逗号分隔的数字 (例如: 10,20): ";
    cin >> a >> comma >> b;

    cout << "第一个数字: " << a << endl;
    cout << "第二个数字: " << b << endl;

    return 0;
}

实践应用

示例1：个人信息录入系统

创建一个简单的个人信息录入程序：

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    int age;
    double height, weight;

    cout << "=== 个人信息录入系统 ===" << endl;
    cout << "请输入您的姓名: ";
    getline(cin, name);

    cout << "请输入您的年龄: ";
    cin >> age;

    cout << "请输入您的身高(米): ";
    cin >> height;

    cout << "请输入您的体重(公斤): ";
    cin >> weight;

    cout << "\n=== 录入完成 ===" << endl;
    cout << "姓名: " << name << endl;
    cout << "年龄: " << age << " 岁" << endl;
    cout << "身高: " << height << " 米" << endl;
    cout << "体重: " << weight << " 公斤" << endl;

    return 0;
}

示例2：简单计算器

使用输入输出实现一个简单的计算器：

#include <iostream>
using namespace std;

int main() {
    double num1, num2;
    char operation;

    cout << "=== 简单计算器 ===" << endl;
    cout << "请输入第一个数字: ";
    cin >> num1;

    cout << "请输入运算符 (+, -, *, /): ";
    cin >> operation;

    cout << "请输入第二个数字: ";
    cin >> num2;

    cout << "\n计算结果: ";
    // 使用if语句判断运算符类型
    if (operation == '+') {
        cout << num1 << " + " << num2 << " = " << (num1 + num2) << endl;
    } else if (operation == '-') {
        cout << num1 << " - " << num2 << " = " << (num1 - num2) << endl;
    } else if (operation == '*') {
        cout << num1 << " * " << num2 << " = " << (num1 * num2) << endl;
    } else if (operation == '/') {
        if (num2 != 0) {
            cout << num1 << " / " << num2 << " = " << (num1 / num2) << endl;
        } else {
            cout << "错误：除数不能为零！" << endl;
        }
    } else {
        cout << "错误：不支持的运算符！" << endl;
    }

    return 0;
}

示例3：成绩统计表

使用格式化输出创建一个成绩统计表：

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    cout << "===============================" << endl;
    cout << " 学生成绩统计表 " << endl;
    cout << "===============================" << endl;

    // 设置列标题
    cout << left << setw(10) << "姓名"
            << right << setw(8) << "语文"
            << setw(8) << "数学"
            << setw(8) << "英语"
            << setw(8) << "平均分" << endl;

    cout << "-------------------------------" << endl;

    // 输出学生数据
    cout << left << setw(10) << "小明"
            << right << setw(8) << 85
            << setw(8) << 92
            << setw(8) << 78
            << setw(8) << fixed << setprecision(1) << (85+92+78)/3.0 << endl;

    cout << left << setw(10) << "小红"
            << right << setw(8) << 90
            << setw(8) << 88
            << setw(8) << 95
            << setw(8) << fixed << setprecision(1) << (90+88+95)/3.0 << endl;

    cout << left << setw(10) << "小刚"
            << right << setw(8) << 78
            << setw(8) << 85
            << setw(8) << 82
            << setw(8) << fixed << setprecision(1) << (78+85+82)/3.0 << endl;

    cout << "===============================" << endl;

    return 0;
}
