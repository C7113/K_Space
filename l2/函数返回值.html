为什么要使用函数？

代码复用与模块化

函数是将一段完成特定任务的代码封装起来，使其可以被多次调用，避免重复编写相同代码，提高程序的可读性和可维护性。

有返回值 vs 无返回值

函数可以分为两类：

  • 有返回值函数：执行后返回一个结果，如同自动贩卖机投入硬币后吐出饮料。
  • 无返回值函数（void）：执行一系列操作但不返回结果，如打印信息。

素数判断示例

通过函数封装素数判断逻辑，便于在不同场景中复用。

bool isPrime(int num) {
    if (num < 2) return false;
    for (int i = 2; i * i <= num; i++) {
        if (num % i == 0) return false;
    }
    return true;
}

函数的定义、声明与调用

函数定义语法

返回类型 函数名(参数列表) { 函数体 }

// 示例：返回两个数中的较大值
int getMax(int a, int b) {
    return a > b ? a : b;
}

函数声明

在调用函数前，需先声明其存在（通常在main函数前或头文件中）。

int getMax(int a, int b); // 函数声明

函数调用

在main函数或其他函数中通过函数名和实参调用。

int main() {
    int result = getMax(4, 5);
    cout << result << endl; // 输出 5
    return 0;
}

函数设计原则

单一职责

一个函数只完成一个明确的任务，如“计算总分”、“判断素数”等。

参数与返回值设计

合理设计输入（参数）和输出（返回值），使函数接口清晰。

🎯 项目实战：函数应用

项目1：篮球比赛得分计算

计算球员总得分：罚球x次(1分)，两分球y次(2分)，三分球z次(3分)。

#include <iostream>
using namespace std;

int calculateScore(int x, int y, int z) {
    return x * 1 + y * 2 + z * 3;
}

int main() {
    int x, y, z;
    cin >> x >> y >> z;
    cout << calculateScore(x, y, z) << endl;
    return 0;
}

项目2：游戏金币计算

根据任务难度(1/2/3)计算金币(100/200/300)，累加总金币。

int getCoins(int level) {
    if (level == 1) return 100;
    if (level == 2) return 200;
    if (level == 3) return 300;
    return 0;
}

int main() {
    int n, total = 0;
    cin >> n;
    for (int i = 0; i < n; i++) {
        int level; cin >> level;
        total += getCoins(level);
    }
    cout << total << endl;
    return 0;
}

项目3：游戏角色属性提升

根据任务等级("low"/"mid"/"high")提升生命、攻击、防御属性。

void updateStats(int& hp, int& atk, int& def, string level) {
    if (level == "low") {
        hp += 10; atk += 5; def += 3;
    } else if (level == "mid") {
        hp += 20; atk += 10; def += 6;
    } else if (level == "high") {
        hp += 30; atk += 15; def += 9;
    }
}

int main() {
    int hp, atk, def, n;
    cin >> hp >> atk >> def >> n;
    for (int i = 0; i < n; i++) {
        string level; cin >> level;
        updateStats(hp, atk, def, level);
    }
    cout << hp << " " << atk << " " << def << endl;
    return 0;
}

项目4：素数回文个数

求11到n之间既是素数又是回文数的个数。

bool isPrime(int num) { /* 如上 */ }

bool isPalindrome(int num) {
    int original = num, reversed = 0;
    while (num > 0) {
        reversed = reversed * 10 + num % 10;
        num /= 10;
    }
    return original == reversed;
}

int main() {
    int n, count = 0;
    cin >> n;
    for (int i = 11; i <= n; i++) {
        if (isPrime(i) && isPalindrome(i)) {
            count++;
        }
    }
    cout << count << endl;
    return 0;
}
