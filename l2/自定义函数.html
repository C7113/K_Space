函数的本质与作用

数学中的函数

在数学中，如果对于任意一个自变量 x 都有唯一确定的因变量 y 与之对应，则称 y 是 x 的函数。这种“输入→处理→输出”的思想正是编程中函数的核心。

编程中的函数

函数是一个实现特定功能的代码块，也称为“子程序”。它有入口（参数）和出口（返回值），通过调用实现代码复用，让程序结构更清晰。

生活中的类比

想象一个“炒菜机”：你放入食材（实参），机器进行处理（函数体），最后输出菜肴（返回值）。不同的食材（参数）可以做出不同的菜（结果）。

形参、实参与函数调用

形式参数（形参）

定义函数时使用的参数，是虚拟变量，用于接收调用时传入的值。

void printSquare(int n) {
    cout << n * n << endl;
}

这里的 n 就是形参。

实际参数（实参）

调用函数时传入的具体值，可以是常量、变量、表达式等。

int main() {
    int x = 5;
    printSquare(x); // x 是实参
    printSquare(10); // 10 是实参
    return 0;
}

调用时，实参的值会传递给形参。

函数声明与定义位置

若自定义函数写在 main() 函数之后，需在上方先进行声明。

// 函数声明
int add(int a, int b);

int main() {
    cout << add(3, 4);
    return 0;
}

// 函数定义
int add(int a, int b) {
    return a + b;
}

🎯 典型项目代码示例

1. 完全数判断

找出 N 以内的所有完全数（因子之和等于自身）

#include <iostream>
using namespace std;

// 判断是否为完全数
bool isPerfect(int n) {
    int sum = 0;
    for (int i = 1; i < n; i++) {
        if (n % i == 0) sum += i;
    }
    return sum == n;
}

int main() {
    int n; cin >> n;
    for (int i = 1; i <= n; i++) {
        if (isPerfect(i)) {
            cout << i << " :";
            for (int j = 1; j < i; j++) {
                if (i % j == 0) cout << " " << j;
            }
            cout << endl;
        }
    }
    return 0;
}

输入：10

输出：6 : 1 2 3

2. 统计数字7的出现次数

统计 [L, R] 范围内所有整数中数字7出现的总次数

#include <iostream>
using namespace std;

// 统计一个数中7的个数
int countSeven(int n) {
    int cnt = 0;
    while (n) {
        if (n % 10 == 7) cnt++;
        n /= 10;
    }
    return cnt;
}

int main() {
    int L, R, total = 0;
    cin >> L >> R;
    for (int i = L; i <= R; i++) {
        total += countSeven(i);
    }
    cout << total;
    return 0;
}

输入：60 80

输出：12

3. 素数回文数个数

求 11 到 n 之间既是素数又是回文数的整数个数

#include <iostream>
using namespace std;

// 判断是否为素数
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}

// 判断是否为回文数
bool isPalindrome(int n) {
    int original = n, reversed = 0;
    while (n) {
        reversed = reversed * 10 + n % 10;
        n /= 10;
    }
    return original == reversed;
}

int main() {
    int n, cnt = 0;
    cin >> n;
    for (int i = 11; i <= n; i++) {
        if (isPrime(i) && isPalindrome(i)) cnt++;
    }
    cout << cnt;
    return 0;
}

输入：23

输出：1
