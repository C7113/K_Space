🔧 C++基础语法核心

数据类型 与 常量

C++中基本数据类型决定了变量的存储方式和取值范围。

类型说明示例
int整型7, -3, 0
double双精度浮点型（默认）7.0, 3.14159
float单精度浮点型7.0f
bool布尔型true, false
char字符型'A', '1'

💡 常量类型判断

常量 7.0 默认是 double 类型，不是 float 或 int。

变量 与 标识符

变量是程序中用于存储数据的命名内存位置。

  • 变量必须先定义后使用
  • 变量名必须是合法的标识符
  • 标识符规则：
    • 以字母或下划线开头
    • 只能包含字母、数字、下划线
    • 不能是关键字

✅ 合法标识符

x321, _x321, x321_

❌ 非法标识符

0x321（以数字开头）

关键字

关键字是C++语言保留的、具有特殊含义的单词，不能用作标识符。

常见关键字：int, for, do, if, else, while, switch, case, break, return 等。

⚠️ 注意

cout 是标准库中的对象，不是C++语言的关键字。

⚙️ 控制结构详解

if 语句

用于条件判断，控制程序分支执行。

if (condition) { // 条件为真时执行 } else { // 条件为假时执行 }

前置/后置运算符

--a 先减1再使用，b++ 先使用再加1。

例如：if (--a > b++) 中，a 先减1，b 使用原值比较，之后再加1。

switch 语句

多分支选择结构，常用于枚举值或固定选项。

  • case 后必须是常量表达式
  • 通常使用 break 防止“穿透”
  • default 子句的 break 可省略（但建议保留）
  • case 后可以没有语句（空分支）

📌 重要规则

各 case 和 default 子句的顺序不影响结果（只要包含 break）。

逻辑运算符

运算符说明示例
&&逻辑与（AND）true && false → false
||逻辑或（OR）true || false → true
!逻辑非（NOT）!true → false

💡 短路求值

&&：左操作数为假则不计算右操作数
||：左操作数为真则不计算右操作数

🔁 循环结构精讲

for 循环

用于已知循环次数的场景。

for (int i = 0; i < n; i++) { // 循环体 }

循环次数分析

for (int i = 6; i < 0; --i) 的循环条件 i < 0 初始不成立，循环0次，与其他循环不同。

while 循环

在条件为真时重复执行循环体。

while (condition) { // 循环体 }

📌 经典应用：累加奇数

计算1到9的奇数和：

int n = 1, sum = 0;
while (n < 10) {
if (n % 2 != 0) sum += n;
++n;
}
// sum = 1+3+5+7+9 = 25

嵌套循环

循环中包含另一个循环，常用于二维操作。

输出 * 的数量

外层循环3次，内层循环4次，共输出 3 * 4 = 12 个 '*'。

📦 数组与字符串

一维数组

  • 在内存中连续存放
  • 长度定义后不可动态扩充
  • 下标从0开始，最后一个元素下标为 长度-1
  • 元素类型必须相同

内存地址计算

首地址为A，每个元素占2字节，则第5个元素 a[4] 的地址为 A + 4 * 2 = A + 8。

二维数组

逻辑上是表格，内存中按行连续存储。

int a[5][10]; // 5行10列

二维数组输入

正确方式是双重循环，逐个输入元素：

for(int i = 0; i < 5; i++)
for(int j = 0; j < 10; j++)
cin >> a[i][j];

字符串处理

C风格字符串以 '\0' 结尾的字符数组。

  • "A" 是字符串（含2字符：'A', '\0'）
  • 'A' 是单个字符
  • 不能用 = 赋值字符串数组
  • 拼接使用 strcat()，复制使用 strcpy()

字符串拼接示例

char a[4] = "ABC", b[4] = "123";
char d[8] = "";
strcat(d, a); // d = "ABC"
strcat(d, b); // d = "ABC123"
cout << d; // 输出 ABC123

🧩 函数与算法

函数基础

  • 每个C++程序必须有且只有一个 main() 函数
  • 程序从 main() 开始执行
  • 函数可以返回 void（无返回值）
  • 调用函数时，实参与形参的个数、顺序、类型应一致
  • 参数传递方式不全是“按值传递”，还有“按引用传递”

变量交换

交换两个变量的值是经典问题。

// 正确方法1：使用临时变量
int tmp = a;
a = b;
b = tmp;

// 正确方法2：使用标准库函数
swap(a, b);

❌ 常见错误

a = b; b = a; 会导致两个变量都变成b的值。

经典算法

冒泡排序

每趟将未排序部分的最大值“冒泡”到末尾。

对 5,6,2,4,3,1,7 排序：

  1. 第一趟后：...6,7
  2. 第二趟后：...5,6,7

最小公倍数 (LCM)

12 和 8 的最小公倍数是 24。

计算方法：LCM(a,b) = (a * b) / GCD(a,b)

📊 总结与备考建议

核心考点总结

  1. 基础扎实：数据类型、变量、标识符、关键字是基石。
  2. 逻辑清晰：if、switch、循环的执行流程必须熟练。
  3. 细节把控：前置/后置运算符、短路求值、内存地址计算等易错点。
  4. 数组理解：内存布局、下标、字符串操作是重点。
  5. 算法思维：掌握冒泡排序、最小公倍数等基础算法。

备考建议

  • 熟练掌握C++基本语法，多写代码验证。
  • 重点练习控制结构和循环的嵌套使用。
  • 理解数组在内存中的存储方式。
  • 动手实现交换变量、排序等经典算法。
  • 注意区分C++与C的输入输出（cin/cout vs scanf/printf）。
  • 仔细审题，特别是“不正确”、“不能”等否定词。
