知识点概览

  • 单分支结构 (if)：根据条件执行代码块
  • 双分支结构 (if-else)：根据条件在两个代码块中选择执行
  • 多分支结构 (if-elif-else)：根据多个条件选择执行不同的代码块
  • 条件表达式：比较运算符、逻辑运算符的使用
  • 嵌套分支：在分支结构内部使用其他分支结构

知识点详解

1. 单分支结构 (if)

单分支结构是最简单的条件控制结构，只有在条件为真时才执行代码块。

基本语法

if 条件:
    语句块

要点：

  • 条件为真(True)时执行语句块
  • 条件为假(False)时跳过语句块
  • 语句块需要缩进(通常4个空格)

示例：判断成绩是否合格

# 判断成绩是否合格
score = float(input("请输入成绩："))
if score >= 60:
    print("恭喜你，成绩合格！")
    print("继续努力，争取更好成绩！")
print("程序结束")

运行结果（输入75）：

请输入成绩：75
恭喜你，成绩合格！
继续努力，争取更好成绩！
程序结束

2. 双分支结构 (if-else)

双分支结构根据条件的真假选择执行两个代码块中的一个。

基本语法

if 条件:
    语句块1
else:
    语句块2

要点：

  • 条件为真时执行语句块1
  • 条件为假时执行语句块2
  • 两个语句块都需要缩进

示例：判断奇偶数

# 判断奇偶数
number = int(input("请输入一个整数："))
if number % 2 == 0:
    print(f"{number} 是偶数")
else:
    print(f"{number} 是奇数")
print("程序结束")

运行结果（输入8）：

请输入一个整数：8
8 是偶数
程序结束

3. 多分支结构 (if-elif-else)

多分支结构可以根据多个条件选择执行不同的代码块。

基本语法

if 条件1:
    语句块1
elif 条件2:
    语句块2
elif 条件3:
    语句块3
...
else:
    语句块N

要点：

  • 依次检查条件，执行第一个为真的条件对应的语句块
  • 如果没有条件为真，则执行else语句块（如果有）
  • elif可以有多个，else最多只有一个且必须在最后

示例：成绩等级评定

# 成绩等级评定
score = float(input("请输入成绩："))

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "E"

print(f"你的成绩等级是：{grade}")

运行结果（输入85）：

请输入成绩：85
你的成绩等级是：B

4. 条件表达式

条件结构依赖于条件表达式，这些表达式的结果是布尔值(True或False)。

比较运算符

运算符 含义 示例
== 等于 5 == 5 结果为 True
!= 不等于 5 != 3 结果为 True
< 小于 3 < 5 结果为 True
<= 小于等于 5 <= 5 结果为 True
> 大于 7 > 5 结果为 True
>= 大于等于 5 >= 5 结果为 True

逻辑运算符

运算符 含义 示例
and 与运算 5 > 3 and 2 < 4 结果为 True
or 或运算 5 > 3 or 2 > 4 结果为 True
not 非运算 not (5 < 3) 结果为 True

5. 嵌套分支

在一个分支结构内部可以包含另一个分支结构，这就是嵌套分支。

示例：登录验证系统

# 登录验证系统
username = input("请输入用户名：")
password = input("请输入密码：")

if username == "admin":
    if password == "123456":
        print("登录成功！欢迎管理员！")
    else:
        print("密码错误！")
else:
    print("用户名不存在！")

课堂练习

练习1：闰年判断

编写一个程序，判断用户输入的年份是否为闰年。

闰年判断规则：

  • 能被4整除但不能被100整除的年份是闰年
  • 能被400整除的年份是闰年
参考代码 
# 闰年判断
year = int(input("请输入年份："))

if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year}年是闰年")
else:
    print(f"{year}年不是闰年")

练习2：BMI指数计算

编写一个程序，根据用户输入的身高和体重计算BMI指数，并给出健康建议。

BMI计算公式：BMI = 体重(kg) / 身高(m)²

健康标准：

  • BMI < 18.5：偏瘦
  • 18.5 ≤ BMI < 24：正常
  • 24 ≤ BMI < 28：偏胖
  • BMI ≥ 28：肥胖
参考代码 
# BMI指数计算
height = float(input("请输入身高(米)："))
weight = float(input("请输入体重(公斤)："))

bmi = weight / (height ** 2)
print(f"您的BMI指数是：{bmi:.2f}")

if bmi < 18.5:
    print("您的体重偏瘦，请注意营养")
elif bmi < 24:
    print("您的体重正常，继续保持")
elif bmi < 28:
    print("您的体重偏胖，请注意锻炼")
else:
    print("您的体重肥胖，请注意减肥")
