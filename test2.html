知识点详解

1. Python常见开发环境介绍

Python是一门易于学习且功能强大的编程语言，在不同的场景下我们可以选择不同的开发环境：

IDLE

IDLE是Python自带的集成开发环境，适合初学者使用。它包含了代码编辑器、调试器和Python Shell。

  • 优点：安装Python时自动安装，无需额外配置
  • 缺点：功能相对简单，不适合大型项目

Visual Studio Code

微软开发的免费代码编辑器，通过安装Python插件可以很好地支持Python开发。

  • 优点：轻量级、插件丰富、界面美观
  • 缺点：需要手动安装和配置插件

Jupyter Notebook

基于Web的交互式开发环境，特别适合数据分析和机器学习领域。

  • 优点：支持实时代码执行、可视化展示、文档编写
  • 缺点：不适合开发大型应用程序

2. IDLE的两种工作模式

Shell模式（交互式）

Shell模式允许我们逐行输入并立即执行Python代码，非常适合测试小段代码或学习语法。

>>> print("Hello, World!") Hello, World! >>> 2 + 3 5 >>> x = 10 >>> x * 2 20

特点：

  • 输入一行代码立即执行
  • 以>>>提示符开头
  • 适合快速测试和实验

File模式（脚本式）

File模式用于编写完整的Python程序，保存为.py文件后可以多次运行。

# hello.py name = input("请输入您的姓名：") print("您好，", name)

特点：

  • 编写完整程序
  • 可以保存和重复运行
  • 适合开发正式的程序

3. IDLE基本操作

在IDLE中开发Python程序需要掌握以下基本操作：

操作 快捷键 说明
新建文件 Ctrl+N 创建新的Python文件
保存文件 Ctrl+S 保存当前编辑的文件
运行程序 F5 运行当前Python文件

4. 文件命名规范

良好的文件命名有助于项目的维护和团队协作：

正确的命名方式

  • calculator.py
  • hello_world.py
  • math_operations_v1.py
  • student_001.py

错误的命名方式

  • 计算器.py（包含中文）
  • my program.py（包含空格）
  • 123test.py（以数字开头）
  • my-program.py（包含特殊字符）

5. Python版本选择

目前Python有两个主要版本：Python 2.x 和 Python 3.x。由于Python 2.x已经停止维护，我们强烈推荐使用Python 3.5.2或更高版本。

推荐版本

Python 3.5.2及以上版本具有更好的性能和更多的特性支持。

练习2：文件命名规范实践

判断下列文件名是否符合Python命名规范：

  1. my_first_program.py
  2. 第一个程序.py
  3. program 001.py
  4. _test.py
  5. 5calc.py
  6. math-calculator.py
点击查看答案

正确的文件名：my_first_program.py、_test.py

错误的文件名及原因：

  • 第一个程序.py - 包含中文字符
  • program 001.py - 包含空格
  • 5calc.py - 以数字开头
  • math-calculator.py - 包含连字符(-)
