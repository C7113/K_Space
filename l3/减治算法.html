假设你面前有12枚硬币，其中有一枚是假币。假币和真币的重量不同，但你不知道假币比真币轻还是重。你有一个天平，但只能使用三次。如何找出这枚假币？

💡 减治算法的核心思想是：逐步减小问题规模，将复杂问题转化为简单子问题解决。

📚 减治算法思想

什么是减治算法？

减治算法通过逐步减小问题规模，将复杂问题转化为简单子问题解决。

减治算法的关键步骤

  1. 逐步减小问题规模：每次处理一个子问题，直到问题易于求解
  2. 解决简化后的子问题：获取中间结果
  3. 合并结果：得到原问题的解

减治算法的优势

  • ✅ 降低问题复杂度，减少计算量
  • ✅ 适用于多种问题，如查找、排序、数值计算等
  • ✅ 提高算法效率，优化程序性能
💡 典型应用：
  • 二分查找：每次将搜索范围减半
  • 阶乘计算：n! = n × (n-1)!
  • 高精度加法：从低位逐位相加

🔢 阶乘计算

问题描述

计算一个整数的阶乘，结果可能非常大。例如，计算 20!，结果为 2432902008176640000，普通整数类型无法存储。

算法思想

从 1 开始逐步乘以每个整数，直到目标整数。例如，计算 5!：

1 × 2 = 2
2 × 3 = 6
6 × 4 = 24
24 × 5 = 120

代码实现（递归）

C++
1#include <iostream> 2using namespace std; 3 4// 递归函数计算阶乘 5int factorial(int n) { 6 if (n == 0) { 7 return 1; // 0的阶乘定义为1 8 } else { 9 return n * factorial(n - 1); // 递归调用 10 } 11} 12 13int main() { 14 int number; 15 cin >> number; 16 cout << factorial(number) << endl; 17 return 0; 18}
⚠️ 注意：当n较大时，普通int类型会溢出！需要使用高精度算法（字符串或数组存储）。

🔍 二分查找

问题描述

在有序数组中查找特定元素，要求高效定位。

算法思想

每次将数组分成两半，根据中间值与目标值比较，缩小搜索范围。

💡 例如，查找数字 5，在数组 [1, 3, 5, 7, 9] 中：
第一次：中间值 5 = 目标，直接找到！

代码实现

C++
1int binarySearch(int arr[], int l, int r, int x) { 2 while (l <= r) { 3 int m = l + (r - l) / 2; // 防止溢出 4 if (arr[m] == x) 5 return m; // 找到目标 6 if (arr[m] < x) 7 l = m + 1; // 在右半部分查找 8 else 9 r = m - 1; // 在左半部分查找 10 } 11 return -1; // 未找到 12}
💡 时间复杂度：O(log n)，每次查找范围减半！

➕ 高精度加法

问题描述

计算两个大整数的和，超出普通整数范围。例如，计算两个 100 位数字的和。

算法思想

  1. 从最低位开始逐位相加
  2. 注意进位处理
  3. 最后反转结果字符串

代码实现

C++
1string highPrecisionAdd(string num1, string num2) { 2 if (num1.length() < num2.length()) 3 swap(num1, num2); 4 5 string result = ""; 6 int len1 = num1.length(); 7 int len2 = num2.length(); 8 int diff = len1 - len2; 9 int carry = 0; 10 11 for (int i = len2 - 1; i >= 0; i--) { 12 int sum = (num1[i + diff] - '0') + (num2[i] - '0') + carry; 13 result.push_back((sum % 10) + '0'); 14 carry = sum / 10; 15 } 16 17 for (int i = diff - 1; i >= 0; i--) { 18 int sum = (num1[i] - '0') + carry; 19 result.push_back((sum % 10) + '0'); 20 carry = sum / 10; 21 } 22 23 if (carry > 0) 24 result.push_back(carry + '0'); 25 26 reverse(result.begin(), result.end()); 27 return result; 28}

⚖️ 减治算法 vs 分治算法

减治算法

  • 逐步减小问题规模，每次处理一个子问题
  • 例如：二分查找、高精度加法
  • 分解方式：问题规模逐渐变小

分治算法

  • 将问题分解为多个独立子问题，分别求解
  • 例如：归并排序、快速排序
  • 分解方式：问题分解为多个子问题
✅ 总结：减治是"化繁为简"，分治是"化整为零"
