什么是模拟算法？

定义

模拟算法是一种根据题目描述的规则，一步一步忠实复现整个过程的解题方法。

核心思想

不需要高深技巧，只需“照着题目说的做”，像搭乐高一样，按说明书一步步执行。

适用场景

题目描述了一个明确的过程或规则，要求计算结果或状态，如游戏规则、时间计算、范围判断等。

模拟算法的关键特征

特征 说明
输入 (Input) 题目给定的初始数据，如人数、数量、范围等
处理 (Process) 严格按照题目规则进行模拟计算或状态变化
输出 (Output) 模拟结束后得到的结果，如编号、数值、范围等
明确性 每一步操作都有清晰的定义和规则
有限性 模拟过程在有限步内结束

经典题目解析

1. 吃粽子（轮次模拟）

n个首领轮流吃粽子，第i轮每人吃i个，吃到最后一个的获胜。

输入：3 13

输出：2

思路：按轮次模拟，计算每轮消耗的粽子总数，直到剩余粽子不足一轮。

2. 寻找幸运数（范围交集）

找出所有同学提供范围的交集，即最小确定范围。

输入：
2
3 10
0 7

输出：3 7

思路：交集左边界取最大值，右边界取最小值。

3. 分南瓜（奇偶判断）

判断一个数能否分成两个奇数之和。

输入：6 8 7

输出：1 1 0

规律：一个数能分成两个奇数当且仅当它为大于2的偶数。

🎯 模拟算法实战代码

吃粽子问题 C++ 实现

#include <iostream>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;

    int round = 1; // 当前轮次
    while (true) {
        int eatThisRound = round * n; // 本轮总共要吃的粽子数
        if (k <= eatThisRound) {
            // 最后一个粽子在本轮被吃完
            for (int i = 1; i <= n; i++) {
                if (k <= round) {
                    cout << i << endl;
                    return 0;
                }
                k -= round;
            }
        } else {
            k -= eatThisRound; // 减去本轮吃的粽子
            round++; // 进入下一轮
        }
    }
    return 0;
}

幸运数范围 C++ 实现

#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int n, L, R;
    cin >> n;
    cin >> L >> R; // 第一个范围
    int low = L, high = R;

    for (int i = 1; i < n; i++) {
        cin >> L >> R;
        low = max(low, L);
        high = min(high, R);
    }

    cout << low << " " << high << endl;
    return 0;
}
