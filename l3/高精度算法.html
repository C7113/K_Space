什么是高精度运算？

高精度运算（大整数运算）是指处理超过标准数据类型范围的大整数的运算。在32位系统中，int类型的最大值约为21亿（2^31-1），当需要计算更大的数时，就需要使用高精度算法。

int 类型的范围：
  • int 类型：4个字节，32位
  • 最大值：2,147,483,647（约21亿）
  • 最小值：-2,147,483,648

高精度运算的原理

将大整数用字符串输入，然后逐位拆分到数组中，按位进行运算，模拟竖式计算的过程。

例：计算 1234 + 5678
       1  2  3  4
    +  5  6  7  8
    -------------
       6  9  1  2
    (进位 1)

高精度运算的基本步骤

  1. 字符串输入：用 string 类型读取大整数
  2. 逆序存储：将字符串转换为数组（低位在前，方便进位处理）
  3. 按位运算：逐位进行加/减运算，处理进位/借位
  4. 结果输出：将数组转换为字符串输出（注意处理最高位的进位）

➕ 高精度加法

算法思路

  1. 将两个加数逆序存入数组（个位在前）
  2. 从低位到高位逐位相加
  3. 当前位置结果 = a[i] + b[i] + 进位
  4. 进位 = 结果 / 10，结果 = 结果 % 10
  5. 处理完所有位后，若最高位有进位则单独处理

核心代码

C++ - 高精度加法核心
1// 字符串转数组（逆序存储） 2void getArray(string s, int a[]) { 3 int len = s.size(); 4 for(int i = 0; i < len; i++) { 5 a[i] = s[len - i - 1] - '0'; // 逆序存储 6 } 7} 8 9// 高精度加法 10void bigAdd(string str_a, string str_b, int c[]) { 11 int len_a = str_a.size(), len_b = str_b.size(); 12 int a[1000], b[1000]; 13 14 getArray(str_a, a); 15 getArray(str_b, b); 16 17 int len = max(len_a, len_b); 18 for(int i = 0; i < len; i++) { 19 c[i] += a[i] + b[i]; // 相加 20 c[i + 1] = c[i] / 10; // 进位 21 c[i] = c[i] % 10; // 取余 22 } 23 24 // 处理最高位进位 25 if(c[len] != 0) len++; 26}

➖ 高精度减法

算法思路

  1. 确保被减数大于减数（否则需要处理负号）
  2. 将两个数逆序存入数组
  3. 从低位到高位逐位相减
  4. 如果当前位不够减，向高位借位
  5. 注意处理前导零

核心代码

C++ - 高精度减法核心
1// 高精度减法 2void bigSub(string str_a, string str_b, int c[]) { 3 int a[1000], b[1000]; 4 int len_a = str_a.size(), len_b = str_b.size(); 5 6 getArray(str_a, a); 7 getArray(str_b, b); 8 9 int len = max(len_a, len_b); 10 for(int i = 0; i < len; i++) { 11 if(a[i] < b[i]) { // 需要借位 12 a[i + 1]--; // 向高位借位 13 a[i] += 10; // 当前位加10 14 } 15 c[i] = a[i] - b[i]; // 相减 16 } 17 18 // 去除前导零 19 while(len > 1 && c[len - 1] == 0) { 20 len--; 21 } 22}

题目：高精度减法

题目描述：
高精度减法，求 a - b。a, b 都是不超过240位的非负整数。

输入：
两个非负整数，每行一个。

输出：
一个整数，代表两个整数相减之后的结果。

输入样例：
33333333333333333333333333333333333333333
22222222222222222222222222222222222222222

输出样例：
11111111111111111111111111111111111111111
