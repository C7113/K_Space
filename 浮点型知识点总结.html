中国古代小数的发展

刘徽与《九章算术注》

魏晋时期的数学家刘徽最早提出“微数”概念，即现代的小数。他写道：“微数无名者以为分子，其一退以十为母，其再退以百为母”。

祖冲之与圆周率

南北朝数学家祖冲之将圆周率 π 精确到 3.1415926 ~ 3.1415927 之间，领先世界近千年！

古代小数单位

中国曾用“分、釐、毫、丝、忽、微、纤”表示小数，每级为前一级的十分之一。

例如：3.1416 读作“三又一分四釐一毫六丝”。

C++ 中的浮点型数据类型

数据类型 定义标识符 数值范围
单精度实型 float -3.4E+38 ~ 3.4E+38
双精度实型 double -1.7E+308 ~ 1.7E+308
长双精度实型 long double -3.4E+4932 ~ 3.4E+4932

提示：编程中推荐使用 double，精度更高更安全。

常量定义与小数保留

✅ 定义常量（如 π）

const double PI = 3.1415926;
// 常量名通常大写，值不可修改

✅ 保留小数位数（需包含头文件）

#include <iomanip>
cout << fixed << setprecision(2) << value;
示例： 计算圆面积并保留2位小数
double r = 1.5;
cout << fixed << setprecision(2) << PI * r * r;
// 输出：7.06（四舍五入）

📄 典型C++代码示例

1. 保留12位小数输出

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double x;
    cin >> x;
    cout << fixed << setprecision(12) << x;
    return 0;
}

输入：3.1415926535897932

输出：3.141592653590

2. 买卡片（打折计算）

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double n, m;
    cin >> n >> m;
    double change = n - m * 0.8;
    cout << fixed << setprecision(2) << change;
    return 0;
}

输入：100 50

输出：60.00

3. 计算总分和平均分

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int x, y, z;
    cin >> x >> y >> z;
    int sum = x + y + z;
    double avg = (double)sum / 3;
    cout << sum << endl;
    cout << fixed << setprecision(1) << avg << endl;
    return 0;
}

输入：100 95 91

输出：

286

95.3

4. 买水果（总价计算）

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double x, y, z;
    cin >> x >> y >> z;
    double total = x * 8.5 + y * 5.6 + z * 6.2;
    cout << fixed << setprecision(1) << total;
    return 0;
}

输入：1.2 5.5 3.9

输出：65.2
