计算机语言与C++简介

什么是计算机语言？

计算机语言是人与计算机之间传递信息的媒介，用于编写指令让计算机执行任务。

语言分类

  • 低级语言：机器语言（01代码）、汇编语言（接近硬件）
  • 高级语言：C、C++、Python、Java（更接近人类语言）
  • 专用语言：SQL（数据库）、HTML（网页）、脚本语言

C++ 的特点

C++ 是 C 语言的继承者，既能进行过程化编程，也支持面向对象编程（继承、多态），适用范围广，从系统开发到游戏编程均有应用。

集成开发环境 (IDE)

编写C++程序需要使用IDE（如Code::Blocks、Dev-C++、Visual Studio），它集成了代码编辑、编译、调试等功能。

C++ 程序的基本结构

代码行 作用说明
#include<iostream> 引入输入输出流头文件，使用 cout 和 cin 必需
using namespace std; 使用 std 命名空间，避免每次写 std::cout
int main() 主函数，程序的入口，从这里开始执行
{ ... } 大括号定义函数体或代码块的范围
return 0; 表示程序正常结束，返回操作系统

“Hello, World!” 程序解析

#include <iostream>
using namespace std;

int main() // 程序从这里开始
{
    cout << "Hello, World!"; // 输出文本
    return 0; // 结束程序
}

输出流与换行控制

cout 输出

cout 是 C++ 中用于屏幕输出的对象，属于 iostream 库，使用 << 流插入运算符。

cout << "要输出的内容";

换行符 endl

endl 是一个控制符，用于输出换行并刷新输出缓冲区。

cout << "第一行" << endl;
cout << "第二行";
// 输出：
// 第一行
// 第二行

头文件与命名空间

头文件（如 iostream）包含函数声明和定义，通过 #include 引入。

命名空间（如 std）用于避免命名冲突，多个库中同名函数可通过 std:: 区分。

🎯 项目实战：输出古诗与图形

项目1：输出《送元二使安西》

使用 cout 和 endl 逐行输出古诗。

#include <iostream>
using namespace std;

int main() {
    cout << "送元二使安西" << endl;
    cout << "渭城朝雨浥轻尘" << endl;
    cout << "客舍青青柳色新" << endl;
    cout << "劝君更尽一杯酒" << endl;
    cout << "西出阳关无故人" << endl;
    return 0;
}

项目2：绘制10x18字符图形

根据规律绘制如下图形（以5x7为例）：

ABCDEFG
BABCDEF
CBABCDE
DCBABCD
EDCBABC

规律：每行首字符递增，后续字符为前一行的左移。

可使用循环和字符运算实现10x18的更大图形。
