问题引入

学校委托我们制作一个用于统计班级人数的程序。

💡 如果有5个班级，我们需要定义5个变量：
int class1, class2, class3, class4, class5;

更高效的方法

如果需要统计全校30个班级的人数呢？定义30个变量太麻烦了！

📦 解决方案：使用数组来存储多个相同类型的数据！

📚 什么是数组？

数组的定义

数组是用于存储多个相同类型数据的有序集合。

数组的特点

  • 📏 大小固定：创建时指定大小，不能改变
  • 🔢 类型统一：所有元素必须是相同类型
  • 📍 索引有序：元素按顺序存储，通过索引访问
💡 就像一排编号的柜子，每个柜子有一个编号（索引），里面存放物品（数据）。

数组示例

C++
1// 班级人数统计: classStu[5] 2int classStu[5] = {35, 42, 38, 40, 45}; 3 4classStu[0]; // 第一个元素，值为35 5classStu[1]; // 第二个元素，值为42 6classStu[4]; // 第五个元素，值为45
⚠️ 注意：C++中数组索引从0开始！

📜 如何定义数组？

基本语法

C++
1数据类型 数组名[元素个数];
  • 数据类型：int, double, char 等
  • 数组名：符合命名规则的标识符
  • 元素个数：必须是常量或常量表达式

定义示例

C++
1// 定义一个能存5个整数的数组 2int classStu[5]; 3 4// 定义数组的同时初始化 5int scores[5] = {90, 85, 78, 92, 88}; 6 7// 初始化时可以省略数组长度，系统会自动推断 8int nums[] = {1, 2, 3, 4, 5}; // 长度为5 9 10// 定义浮点型数组 11double prices[3] = {19.9, 29.9, 39.9};
⚠️ 注意：数组大小必须是常量，如 int a[10]; 可以，但 int n=10; int a[n]; 不行！（C++标准不支持）

🔍 如何访问数组元素？

访问语法

C++
1数组名[索引]

索引是一个整数，表示元素的位置（从0开始）。

访问示例

C++
1int classStu[5] = {35, 42, 38, 40, 45}; 2 3// 访问各个元素 4classStu[0]; // 第一个元素，值为35 5classStu[1]; // 第二个元素，值为42 6classStu[4]; // 第五个元素，值为45 7 8// 输出所有元素 9for (int i = 0; i < 5; i++) { 10 cout << classStu[i] << " "; 11}
💡 小技巧：数组长度为n时，索引范围是 0 ~ n-1

✏️ 如何修改数组元素？

赋值语法

C++
1数组名[索引] = 值;

修改示例

C++
1int classStu[5] = {35, 42, 38, 40, 45}; 2 3// 修改第一个班级的人数为50人 4classStu[0] = 50; 5 6// 把第三个元素改为60 7classStu[2] = 60;

完整示例

C++
1#include <iostream> 2using namespace std; 3 4int main() { 5 int classStu[5] = {35, 42, 38, 40, 45}; 6 7 cout << "原始: "; 8 for (int i = 0; i < 5; i++) { 9 cout << classStu[i] << " "; 10 } 11 cout << endl; 12 13 // 修改第3个班级的人数为50 14 classStu[2] = 50; 15 16 cout << "修改后: "; 17 for (int i = 0; i < 5; i++) { 18 cout << classStu[i] << " "; 19 } 20 21 return 0; 22}

🔄 如何遍历数组？

使用循环遍历

通常使用 for 循环来遍历数组的所有元素。

示例：输出成绩

C++
1#include <iostream> 2using namespace std; 3 4int main() { 5 int n; 6 cin >> n; 7 8 int scores[100]; // 定义足够大的数组 9 10 // 输入n个成绩 11 for (int i = 0; i < n; i++) { 12 cin >> scores[i]; 13 } 14 15 // 输出所有成绩 16 for (int i = 0; i < n; i++) { 17 cout << scores[i] << " "; 18 } 19 20 return 0; 21}

实战：统计优秀人数

题目：输入n个人的成绩，统计成绩优秀（≥90分）的人数。
输入：第一行n，第二行n个成绩
输出：优秀人数
C++
1#include <iostream> 2using namespace std; 3 4int main() { 5 int n, count = 0; 6 cin >> n; 7 8 int scores[100]; 9 for (int i = 0; i < n; i++) { 10 cin >> scores[i]; 11 } 12 13 // 统计优秀人数 14 for (int i = 0; i < n; i++) { 15 if (scores[i] >= 90) { 16 count++; 17 } 18 } 19 20 cout << count << endl; 21 22 return 0; 23}

实战：输出奇数和偶数

题目：输入n个整数，分别输出奇数和偶数。
输入：第一行n，第二行n个整数
输出：第一行奇数，第二行偶数
C++
1#include <iostream> 2using namespace std; 3 4int main() { 5 int n; 6 cin >> n; 7 8 int nums[30]; 9 for (int i = 0; i < n; i++) { 10 cin >> nums[i]; 11 } 12 13 // 先输出奇数 14 for (int i = 0; i < n; i++) { 15 if (nums[i] % 2 == 1) { 16 cout << " " << nums[i]; 17 } 18 } 19 cout << endl; 20 21 // 再输出偶数 22 for (int i = 0; i < n; i++) { 23 if (nums[i] % 2 == 0) { 24 cout << " " << nums[i]; 25 } 26 } 27 28 return 0; 29}

🫧 冒泡排序

什么是冒泡排序？

冒泡排序是一种简单的排序算法，通过反复交换相邻元素，把最大的元素"冒泡"到数组末尾。

💡 就像水底的气泡向上冒一样，最大的元素会逐渐移动到数组最后面。

排序过程演示

C++
1// 原始: 5 3 8 1 2 2 3// 第1轮: 3 5 1 2 8 (8冒到最后) 4// 第2轮: 3 1 2 5 8 (5移到前面) 5// 第3轮: 1 2 3 5 8 (3移到前面) 6// 第4轮: 1 2 3 5 8 (已完成)

代码实现

C++
1#include <iostream> 2using namespace std; 3 4int main() { 5 int n = 10; 6 int a[] = {69, 80, 90, 100, 70, 89, 59, 73, 95, 85}; 7 8 // 冒泡排序 9 for (int i = 0; i < n - 1; i++) { 10 for (int j = 0; j < n - 1 - i; j++) { 11 if (a[j] > a[j + 1]) { 12 // 交换相邻元素 13 int temp = a[j]; 14 a[j] = a[j + 1]; 15 a[j + 1] = temp; 16 } 17 } 18 } 19 20 // 输出排序结果 21 for (int i = 0; i < n; i++) { 22 cout << a[i] << " "; 23 } 24 25 return 0; 26}
💡 如何从大到小排序？把 if (a[j] > a[j + 1]) 改成 if (a[j] < a[j + 1]) 即可！
