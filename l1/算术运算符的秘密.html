程序运行结果预测

以下程序的输出结果是多少？为什么？

#include <iostream>
using namespace std;
int main() {
    int a;
    float b = 1.3273;
    a = b;
    cout << a << endl;
    return 0;
}

答案：输出 1。因为赋值时发生了自动类型转换，float 的 1.3273 被转换为 int 类型，小数部分被截断。

普通赋值运算符

自动类型转换

当赋值运算符两边的数据类型不同时，系统会自动将右边的值转换为左边变量的类型。

赋值语句 右侧原始值 左侧变量类型 存储结果
int num1 = 3.14; 3.14 (float) int 3
float num2 = 20; 20 (int) float 20.0
char s1 = 97; 97 (int) char 'a'
int s2 = 'a'; 'a' (char) int 97

嵌套赋值

利用赋值运算符自右向左的结合性，可以实现多个变量同时赋值。

// 嵌套赋值
int a, b, c;
a = b = c = 100; // 等价于 a = (b = (c = 100));

注意：嵌套赋值前，所有变量必须已声明，不能在赋值时声明变量。

复合算术赋值运算符

代码简洁化

复合赋值运算符是算术运算和赋值运算的结合，使代码更简洁。

复合运算 等价形式 变量num结果 (初始100)
num += 1 num = num + 1 101
num -= 2 num = num - 2 98
num *= 3 num = num * 3 300
num /= 4 num = num / 4 25
num %= 5 num = num % 5 0

特殊运算符

小括号运算符 ()

用于改变运算优先级，明确表达式计算顺序。

double res = ((1 + 2) * 3 - (5 - 4)) / 2;
// 计算过程：
// = (3 * 3 - 1) / 2
// = 8 / 2
// = 4

逗号运算符 ,

用于在一条语句中执行多个操作，整个表达式返回最后一个表达式的值。

int a = 1, b = 2, c;
c = (a++, b++, a + b); // a=2, b=3, c=5

// 常见应用场景：
// 1. for循环中初始化多个变量
for(int i = 0, j = 10; i < j; i++, j--) { ... }

// 2. 宏定义中的链式操作
#define MIN_MAX(a, b, min, max) ((min) = (a) < (b) ? (a) : (b)), ((max) = (a) > (b) ? (a) : (b))

// 3. 一行内执行多个赋值
res = num % 10, num /= 10; // 返回 num /= 10 的结果

🎯 项目实战：运算符应用

项目1：机票打折

输入原价和打折率，计算打折后价格并四舍五入到十位。

#include <iostream>
using namespace std;
int main() {
    int x; double z;
    cin >> x >> z;
    int price = (int)(x * z / 10 + 0.5) * 10; // 四舍五入到十位
    cout << price << endl;
    return 0;
}

项目2：数字对调

将三位数的百位与个位对调。

int num, a, b, c;
cin >> num;
a = num / 100; // 百位
b = num / 10 % 10; // 十位
c = num % 10; // 个位
cout << c * 100 + b * 10 + a << endl; // 对调输出

项目3：细胞分裂

细胞每次分裂数量翻倍，求第五次分裂后数量。

int n;
cin >> n;
n *= 32; // 2^5 = 32
cout << n << endl;
