计算机如何处理文字？

计算机底层处理

计算机基于二进制（0和1）处理所有数据。使用二进制的原因：

  • 电路简单：用“开”(1)和“关”(0)表示状态
  • 可靠性高：抗干扰能力强
  • 逻辑运算基础：支持AND、OR、NOT等操作
  • 标准化：统一的硬件和软件基础

ASCII码的诞生

20世纪60年代，为解决不同厂商编码混乱问题，美国国家标准协会（ANSI）制定了ASCII码（美国信息交换标准代码）。

ASCII码定义了128个字符（0-127），包括：

  • 大写/小写字母 A-Z, a-z
  • 数字 0-9
  • 标点符号 !, ?, ., 等
  • 控制字符（如换行符 \n）

ASCII码成为编程和计算机文化的基石。

char 数据类型

基本概念

char 是C++中用于存储单个字符的基本数据类型，通常占用1个字节内存。

char c1 = 'A'; // 存储大写字母A
char c2 = 'a'; // 存储小写字母a
char c3 = '5'; // 存储字符'5'，不是数字5
char c4 = ' '; // 存储空格

字符与整数的转换

字符转整数（获取ASCII码）

将 char 类型的值赋给 int 变量，得到其对应的ASCII码值。

#include <iostream>
using namespace std;
int main() {
    char c;
    int n;
    cin >> c;
    n = c;
    cout << n << endl; // 输出ASCII码
    return 0;
}

输入 'A'，输出 65。

整数转字符

将整数赋给 char 变量，得到其对应的ASCII字符。

int n;
char c;
cin >> n;
c = n;
cout << c << endl;

输入 97，输出 'a'。

项目3：大小写转换

利用ASCII码差值（'a' - 'A' = 32）进行转换。

// 小写转大写
char c;
cin >> c;
cout << (char)(c - 32) << endl;

// 大写转小写
cout << (char)(c + 32) << endl;

项目4：字母后继

输出字母的下一个字母，z后回到a，Z后回到A。

char c;
cin >> c;
if(c == 'z') cout << 'a';
else if(c == 'Z') cout << 'A';
else cout << (char)(c + 1);

🔐 加密算法：凯撒密码

凯撒密码原理

古罗马凯撒将军发明的加密方法，将每个字母在字母表中向前或向后移动固定位数。

例如：每个字母后移4位，'A'→'E', 'B'→'F', ..., 'X'→'B', 'Y'→'C', 'Z'→'D'。

项目：China加密

将 "China" 每个字母后移4位，得到 "Glmre"。

#include <iostream>
using namespace std;
int main() {
    char c1='C', c2='h', c3='i', c4='n', c5='a';
    c1 += 4; c2 += 4; c3 += 4; c4 += 4; c5 += 4;
    cout << c1 << c2 << c3 << c4 << c5 << endl;
    return 0;
}

密码学的基石

虽然凯撒密码现在容易被破解，但它是密码学的鼻祖，为现代加密技术奠定了基础。它体现了“替换加密”的核心思想。
