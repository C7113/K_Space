算术运算与计算机原理

第一台电子计算机 ENIAC

1943年，为满足二战期间弹道计算的迫切需求，美国研发了第一台通用电子计算机ENIAC。它使用约18,000个真空管，重达27吨，每秒可进行5000次加法或400次乘法运算，标志着计算新时代的开启。

算术逻辑单元 (ALU)

计算机的中央处理器（CPU）中有一个核心组件叫算术逻辑单元（ALU），它负责执行所有的算术运算（加、减、乘、除）和逻辑运算，是计算机进行计算的“大脑”。

四则运算回顾

C++ 支持基本的四则运算符：+（加）、-（减）、*（乘）、/（除）、%（模/取余）。

除法运算详解

整数除法

当两个整数相除时，结果会截断小数部分，只保留整数商。

int a = 5;
int b = 2;
int result = a / b; // 结果为 2，不是 2.5

浮点数除法

当至少有一个操作数是浮点数（float 或 double）时，结果会保留小数。

double a = 5.0;
double b = 2.0;
double result = a / b; // 结果为 2.5

模运算（取余）

什么是模运算？

模运算使用 % 运算符，用于计算两个数相除后的余数。数学上表示为 a mod b。

int a = 10;
int b = 3;
int remainder = a % b; // 结果为 1，因为 10 ÷ 3 = 3 余 1

模运算的应用

模运算在编程中用途广泛，例如：

  • 判断奇偶数：if (n % 2 == 0) 表示偶数
  • 判断是否整除：if (n % k == 0)
  • 循环取值、密码学、进制转换等

🎯 项目实战：应用与计算

项目1：带余除法

输入被除数和除数，输出整数商和余数。

#include <iostream>
using namespace std;

int main() {
    int dividend, divisor;
    cin >> dividend >> divisor;
    int quotient = dividend / divisor; // 整数商
    int remainder = dividend % divisor; // 余数
    cout << quotient << " " << remainder << endl;
    return 0;
}

项目2：人民币支付找零

给定金额，计算100、50、20、10、5、1元钞票各需多少张（优先使用大面额）。

#include <iostream>
using namespace std;

int main() {
    int money;
    cin >> money;
    cout << money / 100 << " "; money %= 100;
    cout << money / 50 << " "; money %= 50;
    cout << money / 20 << " "; money %= 20;
    cout << money / 10 << " "; money %= 10;
    cout << money / 5 << " "; money %= 5;
    cout << money << endl;
    return 0;
}

项目3：两位数/三位数求和

输入一个两位或三位数，求各位数字之和。

// 两位数示例：12 -> 1+2=3
int n; cin >> n;
int sum = n / 10 + n % 10;
cout << sum << endl;

// 三位数示例：235 -> 2+3+5=10
int n; cin >> n;
int sum = n / 100 + (n / 10) % 10 + n % 10;
cout << sum << endl;

项目4：倒序输出两位数

输入68，输出86。

int n; cin >> n;
int reversed = (n % 10) * 10 + n / 10;
cout << reversed << endl;

项目5：计算球体体积

公式：V = (4/3) * π * r³，取 π=3.14，结果保留两位小数。

#include <iostream>
#include <iomanip> // 用于设置小数精度
using namespace std;

int main() {
    double r;
    cin >> r;
    double pi = 3.14;
    double volume = (4.0 / 3.0) * pi * r * r * r;
    cout << fixed << setprecision(2) << volume << endl;
    return 0;
}
