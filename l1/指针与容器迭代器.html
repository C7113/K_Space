📌 指针基础

什么是指针？

指针是一种特殊的数据类型，它存储的是另一个变量的内存地址。通过指针，我们可以间接访问和修改目标变量的值。

内存地址概念：
每个变量在内存中都有一个唯一的地址，就像每个房子都有一个门牌号一样。指针就是记录这个"门牌号"的变量。

指针的定义和使用

C++
1#include <iostream> 2using namespace std; 3 4int main() { 5 int a = 20; // 定义整型变量 a 6 int *p; // 定义整型指针 p 7 p = &a; // & 是取地址运算符，将 a 的地址赋给 p 8 9 cout << "a = " << a << endl; // 输出：20 10 cout << "&a = " << &a << endl; // 输出：a 的内存地址 11 cout << "p = " << p << endl; // 输出：p 存储的地址（与 &a 相同） 12 cout << "*p = " << *p << endl; // * 是解引用，输出：20 13 14 return 0; 15}

数组与指针的关系

数组名本身就是一个指针，指向数组的第一个元素。通过数组名可以直接访问数组元素。

重要特性：
sizeof(数组名) 可以统计数组在内存中的总长度和占用空间。

数组的两种访问方式

方式一：下标法 - 使用数组下标直接访问元素

C++ - 下标法
1#include <bits/stdc++.h> 2using namespace std; 3 4int main() { 5 int a[10]; 6 for(int i = 0; i <= 9; i++) { 7 a[i] = i; // 使用下标赋值 8 } 9 for(int i = 0; i <= 9; i++) { 10 cout << a[i] << " "; // 使用下标访问 11 } 12 return 0; 13}

方式二：指针法 - 使用指针运算访问元素

C++ - 指针法
1#include <bits/stdc++.h> 2using namespace std; 3 4int main() { 5 int a[10]; 6 int *p = a; // 指针 p 指向数组首元素 7 for(int i = 0; i <= 9; i++) { 8 a[i] = i; 9 } 10 for(int i = 0; i <= 9; i++) { 11 cout << *(a + i) << " "; // 指针运算访问 12 } 13 return 0; 14}

🔄 容器迭代器

什么是迭代器？

迭代器(Iterator)是访问容器的统一接口，它的作用是提供一个遍历容器内部所有元素的接口。设计人员无需关心容器内存分配的实现细节。

迭代器的特点：
  • 迭代器类似于指针的对象
  • 能够用来遍历C++标准模板库容器中的元素
  • 每个迭代器对象代表容器中的确定地址
  • 将指针封装，自动管理内存

vector 常用语法回顾

C++
1#include <vector> 2 3// 创建 vector 4vector<int> v; // 空 vector 5vector<int> v(n); // n 个元素的 vector 6vector<int> v(n, 0); // n 个初始值为 0 的元素 7 8// 常用操作 9v.push_back(x); // 尾部添加元素 10v.pop_back(); // 删除尾部元素 11v.insert(it, x); // 在迭代器位置插入元素 12v.erase(it); // 删除迭代器位置的元素 13v.size(); // 获取元素个数 14v.empty(); // 判断是否为空 15v.clear(); // 清空所有元素

迭代器的类型

C++ STL 提供两种迭代器：

类型 说明 函数
正向迭代器 从前往后遍历 begin(), end()
反向迭代器 从后往前遍历 rbegin(), rend()

创建和使用迭代器

C++
1#include <bits/stdc++.h> 2#include <vector> 3using namespace std; 4 5int main() { 6 vector<int> v; 7 for(int i = 0; i <= 9; i++) { 8 v.push_back(i); 9 } 10 11 // 创建正向迭代器 12 vector<int>::iterator myit = v.begin(); 13 14 // 使用指针运算访问元素 15 for(int i = 0; i <= 9; i++) { 16 cout << *(myit + i) << " "; 17 } 18 19 // 另一种遍历方式 20 for(auto it = v.begin(); it != v.end(); it++) { 21 cout << *it << " "; 22 } 23 24 // 使用反向迭代器 25 for(auto it = v.rbegin(); it != v.rend(); it++) { 26 cout << *it << " "; // 输出：9 8 7 6 5 4 3 2 1 0 27 } 28 29 return 0; 30}

📊 项目一：求和（排除最大值）

题目描述

输出一个整数数列中不与最大数相同的数字之和

输入：
输入两行：第一行为 N 为接下来数的个数（ N < 100）
第二行为 N 个整数，数与数之间以一个空格分开，每个整数的范围是 -1,000,000 到 1,000,000

输出：
输出为 N 个数中除去最大数其余数字之和

输入样例：
3
1 2 3

输出样例：
3

思路分析

  • 遍历数组，找出最大值
  • 再次遍历，将所有不等于最大值的数累加
  • 注意：可能有多个最大值，只排除所有等于最大值的数

代码实现

C++
1#include <bits/stdc++.h> 2using namespace std; 3 4int main() { 5 int n; 6 cin >> n; 7 8 vector<int> v(n); 9 for(int i = 0; i < n; i++) { 10 cin >> v[i]; 11 } 12 13 // 找出最大值 14 int maxVal = v[0]; 15 for(int i = 1; i < n; i++) { 16 maxVal = max(maxVal, v[i]); 17 } 18 19 // 累加所有不等于最大值的数 20 long long sum = 0; 21 for(int i = 0; i < n; i++) { 22 if(v[i] != maxVal) { 23 sum += v[i]; 24 } 25 } 26 27 cout << sum << endl; 28 return 0; 29}

📊 项目二：序列倒置

题目描述

给定一个正整数序列，请将这个序列倒置后输出

输入：
输入包括一行，即给定的正整数序列，正整数之间用空格隔开，总个数不超过 10^4，每个数的范围在 [1, 10^4]

输出：
包括一行，即倒置后的序列

输入样例：
3 7 2 1

输出样例：
1 2 7 3

思路分析

  • 使用 reverse() 函数直接翻转
  • 或者使用双指针：从两端向中间交换元素
  • 或者使用反向迭代器遍历

代码实现

C++ - 使用 reverse

📊 项目三：高于平均成绩的学生

题目描述

给定 n 个小朋友的数学成绩，求这 n 个小朋友的平均成绩，以及输出所有高于平均成绩的数学成绩（按输入的顺序输出）

输入：
第一行一个整数 n，接下来 n 行每行一个小于等于 100 的整数。n <= 1000

输出：
第一行是 n 个成绩的平均值，保留3位小数。接下来若干行输出高于平均分的成绩，每行一个数。

输入样例：
5
10
20
30
40
60

输出样例：
32.000
40
60

思路分析

  • 先读取所有成绩到 vector 中
  • 计算总成绩和，求平均值
  • 再次遍历，输出所有大于平均成绩的数
  • 注意：使用 fixed << setprecision(3) 控制小数位数

代码实现

C++
1#include <bits/stdc++.h> 2using namespace std; 3 4int main() { 5 int n; 6 cin >> n; 7 8 vector<int> scores(n); 9 double sum = 0; 10 11 // 读取成绩并计算总和 12 for(int i = 0; i < n; i++) { 13 cin >> scores[i]; 14 sum += scores[i]; 15 } 16 17 // 计算平均值 18 double avg = sum / n; 19 20 // 输出平均值，保留3位小数 21 cout << fixed << setprecision(3) << avg << endl; 22 23 // 输出高于平均分的成绩 24 for(int i = 0; i < n; i++) { 25 if(scores[i] > avg) { 26 cout << scores[i] << endl; 27 } 28 } 29 30 return 0; 31}
