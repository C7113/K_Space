🔄 进制转换

其他进制 → 十进制：按权展开法

(1011)₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11
(A)₁₆ = 1×16¹ + 10×16⁰ = 16 + 10 = 26

十进制 → 其他进制：除基取余法

13 ÷ 2 = 6 ... 1

6 ÷ 2 = 3 ... 0

3 ÷ 2 = 1 ... 1

1 ÷ 2 = 0 ... 1

13的二进制：倒序取余 → 1101

二进制 ↔ 八/十六进制：分组法

二进制每3位一组（八进制），每4位一组（十六进制）

1011 0110 → 十六进制：B6

🔢 补码

为什么需要补码？

为了让计算机只用"加法"就能做减法，简化硬件电路，提高运算速度。

三码对比

  • 原码：最高位=符号位，其余=数值
  • 反码：正数=原码，负数=符号位不变，其余按位取反
  • 补码：正数=原码，负数=反码+1
📚 记忆口诀：
正数三码同一形
负数符号永不改
反码只把数值反
补码反码再加一

ASCII码规律

  • '0'-'9'：连续，值48-57
  • 'A'-'Z'：连续，值65-90
  • 'a'-'z'：连续，值97-122
  • 大小写转换：相差32

⚡ 位运算

基本位运算符

  • &：按位与
  • |：按位或
  • ^：按位异或
  • <<：左移
  • >>：右移

常用场景

判断奇偶：
普通：if (n % 2 == 1)
位运算：if (n & 1)

乘以2^k：n << k
除以2^k：n >> k

🎯 位运算技巧

必背技巧：n & (n-1)

作用：消除二进制中最低位的1

n = 12 (1100)
n-1 = 11 (1011)
n & (n-1) = 1000 (8)

统计1的个数

C++
1int countOnes(int n) { 2 int count = 0; 3 while (n > 0) { 4 n = n & (n - 1); // 核心技巧 5 count++; 6 } 7 return count; 8}
⚠️ 易错点：
if (n & 1 == 1) ❌ 优先级错误！
if ((n & 1) == 1) ✅
