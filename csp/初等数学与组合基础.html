📐 加法原理与乘法原理

加法原理

定义：完成一件事有 n 类互斥的方法，每类方法都能独立完成这件事，则总方法数为各类方法数之和。

示例：
从北京去上海，可以坐飞机（3班）、高铁（5班）、轮船（2班），共有多少种出行方式？
解答：3 + 5 + 2 = 10 种

乘法原理

定义：完成一件事需要 n 个步骤，缺一不可，则总方法数为各步骤方法数之积。

示例：
从北京去上海需先在南京转机。北京->南京有3种方式，南京->上海有4种方式，共有多少种出行方式？
解答：3 × 4 = 12 种

🔢 排列与组合

排列 P(n, m)

从 n 个不同元素中取出 m 个元素，排成有序序列的方案数。

公式： P(n, m) = n × (n-1) × ... × (n-m+1) = n! / (n-m)!

组合 C(n, m)

从 n 个不同元素中取出 m 个元素，组成无序集合的方案数。

公式： C(n, m) = n! / (m! × (n-m)!)

性质：
  • C(n, m) = C(n, n-m)
  • C(n, 0) = C(n, n) = 1
  • 杨辉三角递推：C(n, m) = C(n-1, m-1) + C(n-1, m)

计数问题分析

问题：5个人选3个人去参加劳动，有多少种选法？
解答：C(5,3) = 10 种

问题：5个人选3人分别担任班长、副班长、学习委员，有多少种选法？
解答：P(5,3) = 5×4×3 = 60 种

阅读程序题分析

例题：
int count = 0;
for (int i = 1; i <= 5; i++)
    for (int j = i + 1; j <= 5; j++)
        count++;
cout << count;  // 输出什么？
分析：这相当于从5个数中选2个的组合数 C(5,2) = 10
答案：10

🔍 质数判断与筛法

质数的定义

质数（素数）是只能被 1 和自身整除的大于 1 的自然数。

质数判断

核心思想：如果 n 有因子，必然有一个因子 ≤ √n

C++ - 质数判断
1bool isPrime(int n) { 2 if (n <= 1) return false; // 0 和 1 不是质数 3 if (n == 2 || n == 3) return true; 4 if (n % 2 == 0 || n % 3 == 0) return false; 5 6 // 从 5 开始，步长为 6 (检查 6k-1 和 6k+1) 7 for (int i = 5; i * i <= n; i += 6) { 8 if (n % i == 0 || n % (i + 2) == 0) 9 return false; 10 } 11 return true; 12}

埃氏筛法 (Eratosthenes Sieve)

批量判断质数的高效算法：从2开始，把每个质数的倍数都标记为合数。

C++ - 埃氏筛法
1const int MAXN = 1000005; 2bool is_prime[MAXN]; // true 表示是质数 3 4void sieve(int n) { 5 fill(is_prime, is_prime + n + 1, true); 6 is_prime[0] = is_prime[1] = false; 7 8 for (int i = 2; i * i <= n; i++) { 9 if (is_prime[i]) { 10 for (int j = i * i; j <= n; j += i) 11 is_prime[j] = false; // 从 i*i 开始筛 12 } 13 } 14}

📊 最大公约数与最小公倍数

辗转相除法（欧几里得算法）

原理： gcd(a, b) = gcd(b, a % b)

终止条件：当 b = 0 时，gcd(a, 0) = a

复杂度：O(log(min(a, b)))

求最大公约数 GCD

C++ - GCD
1// 递归写法 2long long gcd(long long a, long long b) { 3 return b == 0 ? a : gcd(b, a % b); 4} 5 6// 迭代写法 7long long gcd(long long a, long long b) { 8 while (b != 0) { 9 long long t = a % b; 10 a = b; 11 b = t; 12 } 13 return a; 14}

求最小公倍数 LCM

公式： lcm(a, b) = a × b / gcd(a, b)

⚠️ 易错点：直接 a * b 可能会导致溢出！
正确做法：先算 a / gcd(a, b)，再乘以 b
C++ - LCM（防溢出）
1long long lcm(long long a, long long b) { 2 if (a == 0 || b == 0) return 0; 3 return (a / gcd(a, b)) * b; // 先除后乘，防溢出 4}
