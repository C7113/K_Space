🎯 CSP入门

CSP比赛简介

CSP（计算机软件能力认证）是中国计算机学会主办的软件能力认证考试，分为CSP-J（入门级）和CSP-S（提高级）。

比赛赛制

  • ⚙️ 评测方式：机考，自动评测
  • 📝 题目类型：选择题 + 编程题
  • ⏰ 时间限制：各地区不同
💡 提醒：CSP复赛需要文件读写！不会文件操作 = 输出为空 = 0分！

💾 位与字节

存储单位

  • 位（bit）：最小单位，0 或 1
  • 字节（Byte）：基本单位，1 Byte = 8 bits
  • 1KB = 1024B，1MB = 1024KB，1GB = 1024MB

int类型存储范围

32位int类型的存储范围是：

-2147483648 ~ +2147483647

无符号整数（unsigned int）：0 ~ 4294967295

📁 文件操作

为什么需要文件操作？

没有文件操作 = 程序读不到数据 = 输出为空 = 0分

文件读写方法

C++
1#include <iostream> 2using namespace std; 3 4int main() { 5 // 【关键步骤】文件重定向 6 freopen("input.txt", "r", stdin); // 读入 7 freopen("output.txt", "w", stdout); // 输出 8 9 // 正常写代码... 10 11 return 0; 12}
⚠️ 注意：文件操作在本地测试时不会显示输入输出，需要在评测系统中运行！
