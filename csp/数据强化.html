📊 前缀和

问题场景

有一个长度为 n 的数组，q 次查询，每次查询区间 [l, r] 的元素之和。

前缀和原理

prefix[i] = prefix[i-1] + a[i]

💡 预先计算好"累计总分表"，查询时只需做一次减法。

代码实现

C++
1vector<long long> prefix; 2 3void build(vector<int>& nums) { 4 int n = nums.size(); 5 prefix.resize(n + 1, 0); 6 for (int i = 1; i <= n; i++) { 7 prefix[i] = prefix[i-1] + nums[i-1]; 8 } 9} 10 11// 查询区间 [l, r]（1-based） 12long long query(int l, int r) { 13 return prefix[r] - prefix[l-1]; 14}
💡 核心思想：空间换时间，预处理后每次查询 O(1)

📝 差分数组

问题场景

给数组区间 [l, r] 内的所有元素加上/减去一个值，重复多次操作。

差分原理

  • 定义：diff[i] = a[i] - a[i-1]
  • 修改：diff[l] += val, diff[r+1] -= val
  • 还原：对 diff 做前缀和即可恢复 a

代码实现

C++
1vector<int> diff; 2 3// 区间 [l, r] 加 val 4void update(int l, int r, int val) { 5 diff[l] += val; 6 diff[r + 1] -= val; 7} 8 9// 还原数组 10void restore() { 11 for (int i = 1; i < diff.size(); i++) { 12 diff[i] += diff[i - 1]; 13 } 14}
✅ 应用场景：区间修改 + 区间查询

🪟 滑动窗口

问题场景

有一个长为 n 的序列 a，以及一个大小为 k 的窗口。求每次滑动后窗口中的最小值和最大值。

单调队列原理

  • 队尾维护单调性
  • 求最小值：保持递增队列
  • 求最大值：保持递减队列

代码实现

C++
1// 求滑动窗口最小值 2deque<int> dq; // 存储下标 3 4void get_min(int k) { 5 for (int i = 1; i <= n; i++) { 6 // 维护单调性：弹出比当前大的 7 while (!dq.empty() && a[dq.back()] > a[i]) { 8 dq.pop_back(); 9 } 10 dq.push_back(i); 11 12 // 移除窗口外的元素 13 if (dq.front() <= i - k) dq.pop_front(); 14 15 if (i >= k) { 16 cout << a[dq.front()] << " "; 17 } 18 } 19}
💡 时间复杂度：O(n)，每个元素最多进出队列一次
